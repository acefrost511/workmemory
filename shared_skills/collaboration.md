# 协作规范 - 跨团队共享

## 任务交接格式

### 主Agent派发任务格式
```
## 任务：{任务名称}
### 执行Agent：{agent_id}
### 输入：{相关资料}
### 要求：{输出格式}
### 参考：{相关记忆}
```

### Agent完成任务格式
```
## 完成：{任务名称}
### 输出：{结果}
### 状态：{完成/需修改}
### 建议：{如有}
```

## 记忆传递规则
1. 前置任务完成后，必须传递完整上下文
2. 使用统一的交接模板
3. 关键信息必须包含时间、来源

## 质量检查清单
- [ ] 内容简介300-400字
- [ ] 清单格式，非表格
- [ ] 信息来源可靠
- [ ] 无事实错误
