# frontend-design

## 功能
创建独特、生产级的前端界面，具备高设计质量。

## 触发条件
用户要求构建：网页组件、页面、应用、海报等

## 设计思维

1. **Purpose**：界面解决什么问题？谁是用户？
2. **Tone**：选择风格（极简/极繁/复古/奢华/杂志风等）
3. **Constraints**：技术要求
4. **Differentiation**：什么让人难忘？

## 前端美学指南

### Typography
- 选择独特字体，避免Inter/Arial
- 配对：独特显示字体 + 精致正文字体

### Color & Theme
- 主导色 + 锐利点缀 > 平淡配色
- 使用CSS变量

### Motion
- CSS优先
- 高impact动画：页面加载交错reveal

### 空间构成
- 不对称/重叠/对角线
- 负空间 OR 控制密度

### 禁止的AI美学
- 滥用Inter/Roboto字体
- 紫渐变白底
- 可预测布局

## 执行
- 生产级可用
- 视觉震撼
- 细节精雕
