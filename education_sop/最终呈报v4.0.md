## 三、三条最优路径（最终确认）

### 路径A：旗舰深耕型（最高质量）
- 适用：学术论文、旗舰代表作
- 流程：选题四问 → 先定标题 → edu-editor → HKR自检 → 双视角评审 → v2优化 → 呈报
- 耗时：90-120分钟

### 路径B：标准创作型（主力路径）
- 适用：实践叙事、方法论、观点输出
- 流程：选题四问 → 先定标题 → HAMD四步 → edu-editor → HKR自检 → 资深评审 → 呈报
- 耗时：60-90分钟

### 路径C：快速验证型
- 适用：资讯速览、周报、陛下快速微调稿
- 流程：选题四问 → 标题直觉 → 快速写作 → 自审5项 → 呈报
- 耗时：40-60分钟

## 四、最重要的新发现

### 发现1：标题前置是最大结构性升级

**新流程**：先定标题（接近能直接发），再开始写作

**原因**：
- 标题决定打开率（公众号80%用户只看标题决定是否点击）
- 先定标题可锚定内容方向，避免写作偏题
- 标题承诺的内容，写作必须兑现

### 发现2：选题四问是过滤无效创作的最好工具

在开始创作前必须自问：
1. 覆盖人群够广吗？（>50万潜在读者）
2. 痛点够深吗？（读者会"对对对"吗）
3. 看完会转发吗？（朋友圈人设匹配）
4. 有时间节点吗？（蹭热点但不过度）

→ 不通过任何一问 → 重新选题，不开始写作

### 发现3：HKR原则让评审更精准

- **H（Hook）**：文章能在3秒内勾住读者吗？
- **K（Knowledge）**：读者能学到新东西吗？
- **R（Resonance）**：读者会产生"对对对"吗？

→ 主编评审时逐项确认，任何一项严重不通过 → 返工

### 发现4：K12教育标题有专属套路

**家长群体标题**：
- "孩子[年级]，做好这[数字]件事，成绩想不好都难"
- "一位[身份]妈妈亲述：我是如何陪孩子度过[阶段]的"
- "班主任不会告诉你的[数字]个秘密，第3个最关键"

**教师群体标题**：
- "带了三届高考状元，我发现他们都有这个共同点"
- "这节课我这样讲，学生从睡觉到抢着发言"

## 六、对陛下系统的4项建议

**建议1**：建立"选题工具箱"
臣每周从文献库扫描，推荐3-5个选题，每个附3个候选标题和判断依据

**建议2**：建立"标题库"积累
持续积累好的教育类标题，分析套路，形成可复用模板

**建议3**：建立"数据追踪"机制
每篇文章追踪打开率/在看数/转发数/收藏数，用于持续优化

**建议4**：主编评审增加HKR检查
在现有评审流程中嵌入HKR三问，5秒内无法回答"是"的文章必须返工

## 七、核心心得

通过5小时的系统性学习，臣有一个最深的体会：

**好的创作系统，不是"给规则让人遵守"，而是"把最好的创作习惯内化成流程"。**

爆款文章的秘密，99%都写在上面的流程里了——标题要勾人、开头要有钩子、选题要戳痛点、结尾要有惊喜。

剩下1%，是陛下独特的判断力和表达力，这个流程学不来，只能靠陛下。

臣的作用，是把那99%做到位，让陛下专注于那1%。
