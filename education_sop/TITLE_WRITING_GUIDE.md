## 一、标题的底层逻辑（HAMD视角）

**标题 = 门。门不对，内容再好也没人进来。**（Tim原话）

标题的本质不是"描述文章"，而是**制造认知缺口**——让读者觉得"我知道一些你不知道的事"，从而产生点击动机。

标题的两个功能：
- **打开功能**：让读者愿意点进来
- **筛选功能**：过滤出真正需要这篇文章的人

好标题 = 制造认知缺口 + 给出填充承诺

## 二、教育类公众号标题数据规律

### 2.1 标题长度

| 标准 | 范围 |
|------|------|
| 最佳长度 | 18-25字 |
| 可接受长度 | 19.7-25.84字 |
| 警戒线 | 超过30字会被截断 |
| 重点信息 | 必须放在前15字 |

**教育类标题特点：**
- 教育类标题是六大领域中最长的（平均19.7-25.84字）
- 陈述句为主，直接表达观点
- 几乎不用逻辑连接词（「因为」「所以」「然而」）

### 2.2 教育类标题的七大写法规律

| 规律 | 说明 | 示例 |
|------|------|------|
| **权威背书** | 借用学霸、清华教授、医生等身份 | 「清华教授：学会这5个字」 |
| **数字量化** | 用具体数字降低决策成本 | 「21天养成阅读习惯」「5个技巧」 |
| **结果导向** | 明确告诉读者能得到什么好处 | 「成绩飙升不是梦」「一战上岸」 |
| **精准人群** | 明确目标读者的身份标签 | 「考研党必看」「新手老师收藏」 |
| **痛点直击** | 抓住学习中的难点和困惑 | 「还在手动批作业？」 |
| **反常识冲突** | 制造认知冲突，打破读者预期 | 「AI时代，教师最重要的能力变了」 |
| **悬念制造** | 留有疑问，让读者想点进看 | 「同事追着问我：你哪来这么多时间？」 |

## 三、18种标题类型（教育类适用版）

以下18种标题类型，臣按教育类公众号的适用程度标注了优先级：

### ⭐ 最推荐教育类使用的5种

**① 速成式标题（★★★★★）**
让读者觉得简单易学，强调学习效果和效率
```
「3步学会AI备课，新手老师也能上」
「5分钟搞懂DeepSeek提问技巧」
```

**② 数字式标题（★★★★★）**
用数字让内容更有说服力，降低决策成本
```
「AI进课堂后，老师最该具备这3种能力」
「我用了这1个方法，每周少改10小时作业」
```

**③ 经验式标题（★★★★★）**
分享实战心得，增强同行信任感
```
「亲测3款AI备课工具，这款最适合语文老师」
「我从设计总监转行做老师，这3件事后悔没早知道」
```

**④ 故事式标题（★★★★★）**
增强代入感，用真实经历引发共鸣
```
「我用AI批改作文3周，说说真实感受」
「同事追着问：你批作业哪来这么多时间？」
```

**⑤ 悬念式标题（★★★★）**
勾起读者好奇心，适合知识揭秘类内容
```
「AI时代的形成性评估：一位老师亲测后发现...」
「为什么优秀老师都在用AI批作业？」
```

### ◆ 次推荐教育类使用的5种

**⑥ 对比式标题（★★★★）**
突出差异，引起兴趣，适合教学方法对比
```
「会讲课 vs 会用AI辅助教学，差距有多大？」
「老教师 vs 用AI的老师，学生更喜欢谁？」
```

**⑦ 提问式标题（★★★★）**
激发读者思考，适合引发讨论
```
「当AI比老师更会批作业，教师的价值在哪？」
「AI进课堂后，学生最该学的能力变了吗？」
```

**⑧ 观点式标题（★★★★）**
表达独特立场，引发共鸣和讨论
```
「我反对AI取代教师的3个理由」
「批改作业这件事，AI做得比你好，不代表它更懂教育」
```

**⑨ 专业式标题（★★★）**
精准吸引垂直读者，适合教研向内容
```
「从DOK框架看AI辅助提问的设计逻辑」
「形成性评估的三个层次：基于Wiliam & Black框架的实践指南」
```

**⑩ 情感式标题（★★★）**
引发情绪共鸣，激发传播
```
「我辞掉设计总监去教书，不后悔这3件事」
「AI让我有时间重新看见学生」
```

## 四、爆款标题的四大可复用模块

把爆款标题拆成零件，重新组装，这是最有效的标题生产方法。

### 模块一：主体+方法+结果+权威背书

```
「我用了这个方法瘦了15斤，医生都说科学」
    主体    方法    结果      权威背书

教育类适配：「我用这1个方法批完作文，同事追着要工具」
```

### 模块二：数字量化模块

```
数字降低决策成本。
- 「3个技巧」「5个方法」比「很多技巧」可信
- 职场文用「3」比「10」更可信（数字越大越像营销）
- 教育文用「3-5」最自然

注意：数字要放在标题前半段更有效
```

### 模块三：冲突感模块

```
用反差制造情绪波动，打破常识引发验证欲。

「每天加班到12点，工资却比新人低」
「我劝你别做AI教学」
「当AI批完作业，老师的时间去哪了？」

冲突感的核心：读者预期 vs 实际结果的落差
```

### 模块四：具体行为模块

```
用具体动作触发好奇心，减少抽象概念。

「删了100个好友之后」（比「我开始整理社交圈」更具体）
「批改作文时加了这个步骤，学生主动来复盘」

越具体，越有画面感，越能引发点击
```

## 五、标题避坑指南（5种限流标题）

| 坑 | 错误写法 | 正确写法 |
|----|---------|---------|
| 绝对化用词 | 「最有效」「100%」「必看」 | 「亲测有效」「试了3次都管用」 |
| 标题党 | 「我被开除后月入10万」 | 「从被开除到月入10万，我踩过的3个坑」 |
| 热点乱蹭 | 热点词和内容无关 | 热点必须与文章主题强关联 |
| 过长 | 超过30字 | 重点放前15字，总长18-25字 |
| 名不副实 | 标题承诺内容实现不了 | 标题承诺的内容正文必须兑现 |

**绝对禁止用语：**
- 「必须收藏」「你不得不知道」「专家建议」（居高临下）
- 「重磅」「刚刚」「突发」（标题党）
- 「我觉得」「我认为」（主观感受，缺乏可信度）

## 六、HAMD标题制作标准流程

### 第一步：先写10个备选标题

拿到选题后，不要先写正文，而是先写10个标题备选：

```
格式：[标题内容]（预估类型：痛点型/数字型/反常识型/悬念型/...）
```

**10个备选的要求：**
- 必须覆盖不同类型（不能全是同一种类型）
- 必须包含具体数字或具体场景
- 必须有动词
- 必须回答：读者点进来能得到什么

### 第二步：主编审核通过3-5个

**通过条件：**
- 有价值承诺（读者知道能得到什么）
- 有具体性（不是泛泛而谈）
- 有差异化（和同类文章拉开距离）
- 有动词或情绪钩子

**未通过的标志：**
- 「浅谈AI在教育中的应用」→ 太泛
- 「AI时代教师的挑战与机遇」→ 两边都说，没有立场
- 「教育者必看的AI工具」 → 居高临下+没有利益承诺

### 第三步：选标题写正文

选定的标题必须和内容形成**记忆闭环**：
- 标题承诺的价值 → 正文必须兑现
- 标题设置的悬念 → 正文必须回答
- 标题制造的缺口 → 正文必须填充

### 第四步：标题自检三问

写完正文后回头检查标题：
1. 前15字有没有放重点？（手机端会截断）
2. 有没有承诺读者能带走什么？
3. 正文有没有兑现标题的承诺？

## 七、教育类标题公式手册（直接套用）

### 痛点型（最适合课堂实操内容）
```
「还在手动批作文？1个AI工具3分钟搞定」
「AI辅助教学最常见的3个误区，我踩过」
「用DeepSeek备课，我最后悔没早知道这1点」
```

### 数字干货型（最适合方法论内容）
```
「AI进课堂后，老师最该具备这3种能力」
「批改效率提升80%：我用这5步做到的」
「老师最常用的3款AI工具，亲测推荐」
```

### 反常识型（最适合认知升级内容）
```
「同事追着问：你批作业哪来这么多时间？」
「AI时代，教师最重要的能力不是AI，是提问」
「我辞掉设计总监去教书，最值的选择是这件事」
```

### 悬念故事型（最适合课例分享内容）
```
「我用AI批改作文3周，真实感受（含避坑指南）」
「一堂AI课学生全情投入，我做了这3件事」
「从设计总监到老师，这1年我踩过的坑」
```

### 提问型（最适合引发讨论的内容）
```
「当AI比老师更会批作业，教师的价值在哪？」
「AI时代的形成性评估：到底怎么用才对？」
「老师要不要学AI工具？看这3个判断标准」
```

### 蹭热点型（政策解读/工具发布）
```
「DeepSeek进课堂后，我做了这5件事」
「教育部AI教育通知发布，老师最该关注这3点」
「Sora发布后，我对AI教学的判断变了」
```
