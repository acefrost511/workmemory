# 方案A：稳健深耕型

## 核心流程
```
素材 → edu-editor(v1) → 资深教师评审 → 新手教师评审 → 主编汇总 → v2优化 → 呈报
```

## 方案特征
- **风格**：严谨、稳健、重评审
- **优势**：多视角把关，质量稳定
- **劣势**：环节多，耗时长（约90-120分钟/篇）

## 写作铁律（13条，全部执行）

### 通用铁律
1. 概念不删，但必须配"翻译"（首现术语→口语化解释+具体场景）
2. 每个抽象论点紧跟可落地的教学例子（学科/年级/课堂动作）
3. 数据和来源是硬通货（保留原始数字，注明来源机构）
4. 开头"破题"而非"寒暄"（3句内必须进场景）
5. 结尾必须有独立传播力（金句+可执行最小行动）
6. 语气"尊重型对话"，不居高临下
7. 标题和小标题传递实质信息（好小标题=微型论点）
8. 术语首次出现必须括号注明含义

### 实践叙事类专项
1. 开头要"痛"，不要寒暄（3句内进场景，用具体数字制造共鸣）
2. 数据要"真"，不要堆砌（精确分钟数+参照系）
3. 惊喜有画面感（具体学生行为），烦恼有清单感
4. 结尾有金句，更要有行动（附可直接复制的工作流程）
5. 术语要"着陆"，每千字不超过5个，首现必配解释

### AI/技术工具类专项
1. 必须包含AI典型失误类型（举2-3个具体例子）
2. 必须有教师快速核查方法（5分钟内可完成的检查步骤）
3. 必须标注不可省略的人工环节（"复核清单"）
4. 涉及不同学校条件时，明确说明文章适用场景

## 评分体系

### 资深教师评分（5维度，满分50）
- 专业深度(1-10)、实操价值(1-10)、场景适配(1-10)、专业规范(1-10)、可操作性(1-10)

### 新手教师评分（3维度，满分30）
- 可读性(1-10)、吸引力/标题(1-10)、阅读体验(1-10)

### 主编评分（2维度，满分20）
- 主题契合度(1-10)、传播潜力(1-10)

### 加权计算
实践分享类：资深×25% + 新手×45% + 主编×20% = 满分100

### 双重底线
- 综合底线：加权总分 ≥ 80分
- 新手教师底线：新手教师得分 < 总分33% → 强制返工

## 输出要求
- 字数：1500-2000字
- 格式：Markdown
- 每篇标注：字数、类型、核心观点、读者定位
