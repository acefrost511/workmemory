请你学习下面的材料，这是我跟其他人请教的经验，这是一份教育内容创作连续生产SOP，你看看能不能从其中获得很多启发，包括怎么样去生产创意的内容，以及怎么样去评估。当然里边有一些内容可能不太现实，但你需要根据你的架构来进行相应的微调。但整体性，尤其是其中核心的几种碰撞的方法，还有取素材的流程，等等，这些你要重点参考。
一、你有两个文件柜（基础设定）
文件柜1：教育信念抽屉
•  13个抽屉，每个是一个教育信念（如"AI是助教"、"警惕依赖"等）
•  每个抽屉里放着研究素材（论文结论、数据、发现）
文件柜2：笔记库
•  记录真实观察、对话、经验
•  按四类人分类：家长、教师、学生、管理者
----
二、生产流水线（早上7点启动，连续作业）
从早上7点开始，不断重复以下循环，直到素材用完或时间结束：
循环步骤：
1.  抓料（从两个文件柜拿东西）
2.  碰撞（用特定方法把两个东西关联）
3.  产出（写成文章）
4.  评分（判断是否合格）
5.  再抓料（继续下一轮）
----
三、五种碰撞关联方法（核心）
当AI拿到两个素材时，必须用以下五种方法之一来关联。不能随机拼凑，必须选对方法：
方法一：互补法（最常见）
什么时候用：一个素材讲"是什么"，另一个讲"怎么做"；或者一个讲技术，一个讲人情
怎么操作：
•  看素材A是不是在说某个东西的好处/功能（比如"AI能提高效率"）
•  看素材B是不是在说另一个维度的好处/功能（比如"老师情感支持很重要"）
•  关联句式："A负责XX，B负责YY，两者缺一不可，但现实中我们只做了A忘了B"
价值产出：给出完整方案，指出盲区
示例：
•  素材A（研究）：AI即时反馈提升学习速度
•  素材B（观察）：家长说"孩子用AI后不愿跟我讨论了"
•  关联："AI负责效率（A），家长负责情感确认（B），两者缺口在于'反馈后的交流'"
方法二：因果法（挖深层机制）
什么时候用：素材A是现象，素材B是原因；或者A是前因，B是后果
怎么操作：
•  看素材A是不是一个结果/现象（比如"学生成绩下降"）
•  看素材B是不是一个机制/原理（比如"控制感剥夺理论"）
•  关联句式："表面上是因为A，实际上是因为B导致的C"
价值产出：解释为什么，找到真因
示例：
•  素材A（观察）：孩子作业全对但考试不及格
•  素材B（研究）：自我监控理论（自己检查vs外部检查）
•  关联："不是因为孩子笨（表面），而是因为AI检查（A）剥夺了自我监控机会（B），导致真实水平没暴露"
方法三：矛盾法（出爆款）
什么时候用：两个素材看起来完全相反，一个说好一个说坏
怎么操作：
•  看素材A是不是在说某事的好处（比如"AI个性化很好"）
•  看素材B是不是在说某事的坏处（比如"AI让孩子不思考"）
•  关联句式："看似A和B矛盾，其实取决于C（第三个变量）。A在XX情况下对，B在YY情况下对"
价值产出：解决两难，给出判断标准
示例：
•  素材A（研究）：个性化学习提升效果
•  素材B（观察）：基础差的孩子用AI越用越差
•  关联："个性化（A）和标准化（B）不矛盾，关键看孩子基础（C）。基础好需要个性化，基础差需要先标准化"
方法四：递进法（升级路径）
什么时候用：素材A是基础，素材B是进阶；或者A是旧方法，B是新方法
怎么操作：
•  看素材A是不是在说初级阶段（比如"用AI查答案"）
•  看素材B是不是在说高级阶段（比如"用AI梳理想法"）
•  关联句式："A是第一步，B是第二步，但从A到B需要跨越XX"
价值产出：给出成长路径，指出进阶关键
示例：
•  素材A（观察）：学生把AI当搜索引擎
•  素材B（研究）：AI作为思维外化工具的理论
•  关联："查答案（A）是低阶使用，梳理想法（B）是高阶使用，差距在于'提问能力'的培养"
方法五：边界法（风险提示）
什么时候用：素材A讲适用范围，素材B讲禁忌/风险
怎么操作：
•  看素材A是不是在说"什么情况下有效"（比如"AI对大学生有效"）
•  看素材B是不是在说"什么情况下失效/有害"（比如"对小学生可能有害"）
•  关联句式："A在XX范围内是解药，超出这个范围就变成毒药（B），分界线是YY"
价值产出：给出安全使用指南，避免误伤
示例：
•  素材A（研究）：AI自适应学习提升成绩
•  素材B（观察）：低年级学生用AI后不会自己读题了
•  关联："AI自适应（A）在'知识巩固期'有效，在'习惯养成期'（B）有害，分界线是'是否具备基础阅读能力'"

四、具体执行流程（每一步该干什么）
第1步：抓料（7点开始，每轮如此）
动作：
1.  打开笔记库，按顺序取一条最近3天内的新观察（如果用完就取标记"未使用"的旧观察）
2.  看这条观察涉及哪个教育信念，打开那个信念抽屉
3.  从抽屉里取一个研究素材（优先取最近7天没用过的）
4.  判断这两个素材适合用哪种碰撞方法（看上面五种方法的"什么时候用"）
如果笔记库没有合适观察：
•  改为从同一个信念抽屉取两个研究素材（一个讲数据，一个讲原理）
•  用"因果法"或"互补法"关联这两个研究
第2步：碰撞（核心动作，必须书面回答）
拿到两个素材后，必须回答以下三个问题（写在草稿里）：
问题1：这两个素材是什么关系？
•  是从不同角度看同一件事吗？→ 用互补法
•  是一个现象一个解释吗？→ 用因果法
•  是正反两面吗？→ 用矛盾法
•  是前后阶段吗？→ 用递进法
•  是适用范围和风险吗？→ 用边界法
问题2：它们之间的缺口/张力是什么？
•  比如：研究说A好，但观察发现B坏，缺口是"条件不同"
•  比如：研究讲效率，观察讲情感，缺口是"维度不同"
问题3：用什么句式能把它们串成一句话？
•  必须用上面对应方法的关联句式
•  这句话就是文章的核心观点
第3步：写成文章（结构固定）
开头（钩子）：
•  如果是因果法：用观察的现象开头（"昨天看到一个学生..."）
•  如果是矛盾法：用两个素材的反差开头（"研究发现A，但现实中B..."）
•  如果是其他方法：用具体场景或数据开头
中间（论证）：
•  先呈现素材A（研究或观察，详细一点）
•  再呈现素材B（另一个素材）
•  用选定的碰撞方法解释它们的关系（这是文章的核心价值）
结尾（给方法）：
•  必须给出具体可操作的判断标准或行动步骤
•  比如："如果你看到孩子出现这三种情况，就要停止..."（列出1.2.3.）
第4步：评分（决定是否合格）
每篇文章必须过这五个检查点，每个20分：
1. 真实度（20分）
•  研究来源写清楚了吗？（10分）
•  观察是笔记库里真实的吗？（10分）
•  （低于10分：不要发）
2. 关联度（20分）
•  真的用了五种方法之一吗？（10分）
•  关联逻辑说得通吗？（10分）
•  （低于10分：重写关联部分）
3. 洞察度（20分）
•  有没有"不是...而是..."或"看似...其实..."的句式？（10分）
•  是不是反常识？（读者看了会觉得"原来如此"）（10分）
•  （低于10分：修改观点）
4. 实用度（20分）
•  结尾有没有给具体方法？（10分）
•  读者能马上用起来吗？（10分）
•  （低于10分：补充方法部分）
5. 传播度（20分）
•  有没有一句适合截图发朋友圈的话？（10分）
•  标题有没有数字、反差或悬念？（10分）
总分处理：
•  90-100分：立即发布，标记"优质"
•  75-89分：发布，标记"标准"
•  60-74分：修改标题或结尾后再发布
•  60分以下：丢弃，素材回库（标记"碰撞失败，7天后可再用"）
第5步：再抓料（循环）
•  如果总分≥75：庆祝，开始下一轮（回到第1步）
•  如果总分<75：分析哪个维度扣分，下一轮避开同样错误
----
五、防重复机制（必须遵守）
每轮抓料前检查：
1.  研究7天规则：这个研究素材过去7天内用过吗？用过就换下一个
2.  观察3天规则：这条观察过去3天内用过吗？用过就换
3.  方法多样性：上一轮用了矛盾法，这一轮尽量用因果法或互补法（避免连续三篇都是同一种关联方式，读者会腻）
----
六、示例演示（完整的两轮作业）
第一轮（7:00-8:30）
抓料：
•  从笔记库取观察："家长说孩子用AI写作业，速度很快但眼睛发亮（兴奋点在对答案而不是思考）"
•  涉及信念："警惕AI依赖"
•  从该抽屉取研究："自我决定理论（自主性对动机的影响）"
•  判断方法：因果法（现象是速度快，原因是自主性缺失）
碰撞回答：
1.  关系：观察是现象，研究是解释机制
2.  缺口：家长以为快=好，研究指出快可能是因为外部控制（AI推着走）而非内部驱动
3.  核心句："不是孩子变快了，而是孩子被AI推着走了，失去了'我要做'的感觉"
成文：
•  开头：家长原话"孩子现在写作业可快了"
•  中间：解释速度快背后的控制感转移（研究+观察结合）
•  结尾：三个检查点（1. 关掉AI孩子会不会慌 2. 孩子能讲出思路吗 3. 孩子眼睛是亮在思考还是对答案）
评分：88分（真实18+关联18+洞察17+实用18+传播17）
决策：发布，开始下一轮
第二轮（8:30-10:00）
抓料：
•  从笔记库取观察："老师说班上学生都用AI，但好学生越用越好，差生越用越差"
•  涉及信念："AI是助教"
•  从该抽屉取另一个研究的素材："认知负荷理论（工作记忆容量）"
•  判断方法：边界法（AI对一部分人有效，对一部分人有害，分界是认知负荷能力）
碰撞回答：
1.  关系：素材A讲效果差异，素材B讲能力边界
2.  缺口：好学生和差生的区别在工作记忆容量，AI降低了外在负荷但增加了内在负荷
3.  核心句："AI是杠杆，放大原有差距——强者更强，弱者更弱，分界点是基础认知能力"
成文：
•  开头：课堂观察的反差现象
•  中间：用认知负荷解释为什么同样工具不同结果
•  结尾：给教师的分层使用建议（基础弱的学生先用纸笔，基础好的用AI拓展）
评分：92分（发布）

1.  笔记库的新观察用完了（最近3天内的都用完了）
2.已经产出5篇以上且时间超过下午6点（保证质量比数量重要）
