# 个性化答疑多Agent协作方案

> 研究时间：2026-03-09
> 目标：多个专家Agent协作，从认知、思考、心理、动力等多维度为学生服务

---

## 一、理论基础

### 1. 支架式教学 (Scaffolding) - 维果茨基
- **核心**：最近发展区(ZPD) - 学生现有水平 vs 潜在发展水平
- **应用**：提供临时性的学习支架，随着学生能力提升逐渐撤除

### 2. 苏格拉底问答法
- **核心**：通过提问引导思考，而非直接告知答案
- **应用**：激发批判性思维，让学生在对话中自己发现答案

### 3. 自我效能感理论 - 班杜拉
- **核心**：个体对自己完成某项任务能力的信念
- **应用**：通过成功体验、榜样示范、言语鼓励提升自我效能

### 4. 成长型思维 - 德韦克
- **核心**：能力是可以通过努力和学习来发展的
- **应用**：强调努力和方法，而非天赋和智商

### 5. 认知负荷理论 - Sweller
- **核心**：工作记忆容量有限，需减少外在负荷
- **应用**：简化信息呈现，分步教学，利用图式减少负荷

## 二、协作方案设计

### 方案一：五阶段融合协作模型

```
学生提问 → 并行分析 → 融合策略 → 分步引导 → 效果评估
              ↓
         ┌─────────────────────────────────────┐
         │ 理论专家(What)    - 知识原理       │
         │ 实践专家(How)     - 方法步骤       │
         │ 心理专家(Why)      - 动机情感       │
         │ AI技术专家(Tech)  - 技术支持       │
         │ 评估专家(Check)   - 效果评估       │
         └─────────────────────────────────────┘
```

### 方案二：动态调整模型

根据学生实时反馈，动态调整各专家的参与度：

| 学生状态 | 主导专家 | 辅助专家 |
|----------|----------|----------|
| 理解困难 | 心理专家(降低难度) | 理论专家(简化解释) |
| 动力不足 | 心理专家(激励) | 实践专家(成功体验) |
| 思维卡住 | 理论专家(苏格拉底提问) | AI技术(提示) |
| 理解较好 | 实践专家(变式训练) | 评估专家(检验) |

### 方案三：认知-情感双轨模型

```
         ┌─────────────────┐
         │   问题输入     │
         └────────┬────────┘
                  ↓
    ┌─────────────┴─────────────┐
    │       认知轨道           │  ← 理论+实践+AI
    │  知识理解 → 方法掌握     │
    └─────────────┬─────────────┘
                  ↓
    ┌─────────────┴─────────────┐
    │       情感轨道           │  ← 心理+成长型思维
    │  动机激发 → 信心建立     │
    └─────────────┬─────────────┘
                  ↓
         ┌─────────────────┐
         │   融合输出     │
         └─────────────────┘
```

## 三、具体协作流程

### 第一步：问题诊断（所有专家并行）
- **理论专家**：分析题目涉及的知识点
- **实践专家**：识别解题关键步骤
- **心理专家**：评估学生可能的心理状态（畏难？焦虑？）
- **AI技术专家**：搜索类似题目和解题方法
- **评估专家**：设计理解检测问题

### 第二步：策略融合（输出融合方案）
基于诊断结果，融合各专家建议，形成统一引导方案：
- 引导话术（来自理论+实践）
- 心理支持（来自心理）
- 拓展问题（来自AI+评估）

### 第三步：分步引导（执行融合方案）
- 每次只给一个思考任务
- 观察学生反应
- 根据反馈动态调整

### 第四步：效果评估（实时检测）
- 能否复述原理？
- 能否独立解答变式题？
- 是否有信心面对类似问题？

## 四、测试评估体系

### 评估维度

| 维度 | 指标 | 检测方法 |
|------|------|----------|
| **认知维度** | 概念理解/方法掌握/迁移能力 | 变式题测试/原理复述 |
| **情感维度** | 学习动机/自我效能/抗挫折 | 问卷/访谈 |
| **思维维度** | 批判性/创造性/深度 | 追问检测/问题质量 |

### 评分标准

每维度5分制：
- 5分：优秀 - 能举一反三，主动提问
- 4分：良好 - 能理解并完成同类题
- 3分：及格 - 能理解但需辅助
- 2分：困难 - 仍有明显障碍
- 1分：严重困难 - 需要重新评估

## 五、方案选择建议

| 学生类型 | 推荐方案 |
|----------|----------|
| 基础薄弱 | 方案一 + 心理专家主导 |
| 思维活跃 | 方案二 + 苏格拉底提问 |
| 动力不足 | 方案三 + 情感轨道强化 |
| 综合提升 | 方案一 + 动态调整 |
