## 使用规则（陛下2026-03-30定制·铁律）

**存储时机（每轮对话后立即执行）：**
- 臣每次回复后，分析本轮内容是否能归入现有topics

- 不确定 → 宁可多存也不遗漏

**读取时机（仅在以下情况）：**
- 陛下问"昨天/之前的事"时 → 查TOPIC-INDEX.md → 读对应topics文件
- 陛下问特定话题时 → 直接读对应topics文件
- **当日内容**：不查topics，直接用上下文

**禁止事项：**
- 不要每轮对话都重新读取topics（浪费token）
- 不要在topics里重复抄录已有内容（增量追加即可）
- topics文件命名：/workspace/memory/topics/话题名.md
