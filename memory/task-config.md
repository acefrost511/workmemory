# K12 AI教育资讯追踪任务配置

## 任务模式
日报 + 周报结合

### 日报（每天）
- 时间：早上9点发"内容审核列表"，审完后30分钟内推送正式日报
- 逻辑：有内容就推，没有就不推
- 内容：最近一天内所有资讯（不限于Top 10），去重后全部列出

### 周报（每周六）
- 时间：同样9点发审核列表，审完再发正式周报
- 内容：汇总近一周值得关注的资讯

## 审核流程

### 第一步：AI初审

#### 1. 通用过滤规则
- **时效性硬门槛**：资讯必须在检索周期内（日报近1天，周报近7天）
- **来源可信度校验**：无法确认权威性的来源不放进列表
- **渠道黑名单**：百家号、财富号、小红书、抖音、百度系产品一律排除

#### 2. 新研究板块
- 期刊白名单严格执行，不在清单内的期刊自行核查后排除
- **K12硬边界**：研究对象必须是K12阶段（学前到高中），高等教育/职业教育/成人教育全部排除
- 实践导向优先：过于理论化的研究标"低"优先或不纳入

#### 3. 新产品板块
- 只看国外公司、优先看有融资背景或大公司投资的产品

#### 4. 新报告板块
- 优先收录：UNESCO、OECD、世界银行、各国教育部、顶尖大学研究中心、顶级咨询公司
- 渠道黑名单：百度系、财富号、营销号、未经认证自媒体
- 时效性加严：政策类只看最近发布的，趋势类最多放宽到近一个月

#### 5. 自动标注规则
- "⚠️ 来源待确认"：来源机构不在白名单且无法100%确认权威性
- "🔁 与前期内容相关"：是此前已推送内容的后续进展，标注关联的历史编号
- "🌟 编辑推荐"：综合判断后最值得关注的1-2条

### 第二步：人工审核

**审核列表格式（必须包含）：**
1. 序号
2. 资讯标题
3. 内容简介（300-400字）← 关键
4. 来源机构
5. 建议优先级（高/中/低）
6. 是否需要深度解读（是/否）

**用户回复格式：**
- 通过项: 1,3,5,7
- 不通过: 4,6（理由：与K12关联度低）
- 深度解读: 1,5

---

## 深度解读报告
- 触发：人工审核时用户告知需要单独写解读
- 字数：1500字左右
- 不在日报/周报范围内，单独生成
- 结构：调用 edu-editor 技能写作

---

## 时间安排
- 05:00 ━━ 开始搜索（提前4小时准备）
- 06:30 ━━ 数据处理完成（去重>85%）
- 08:00 ━━ 发送【审核列表】给用户
- ━━━ 用户审核回复（通过项、深度解读需求）
- 09:30+ ━━ 推送【正式报告】（审核完成后30分钟内）
