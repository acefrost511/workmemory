# K12 AI教育资讯追踪任务配置

_最后更新：2026-03-06_

## 工作原则

**事事有回音，件件有回应**
- 完成的任务立即汇报结果
- 未完成的任务每天汇报进展
- 主动学习方法解决问题，不等待用户提醒
- 每天记录用户指导到记忆文件
- 经常自我反思总结经验教训

---

## 一、任务模式

### 日报（每天）
- **时间**：早上9点发"内容审核列表"，审完后30分钟内推送正式日报
- **逻辑**：有内容就推，没有就不推
- **内容**：最近一天内所有资讯（不限于Top 10），去重后全部列出

### 周报（每周六）
- **时间**：同样9点发审核列表，审完再发正式周报
- **内容**：汇总近一周值得关注的资讯

## 三、审核流程（核心）

### 第一步：AI初审（内容运营总监负责）

#### 1. 通用过滤规则

**时效性硬门槛：**
- 日报看近1天
- 周报看近7天
- 超出时间范围一律不进入审核列表

**来源可信度前置校验：**
- 任何进入审核列表的内容，来源渠道必须经过核实
- 无法确认来源权威性的，不放
- 确实存疑但有价值：单独标注"⚠️ 来源待确认"并说明疑点
- **黑名单**：财富号、百家号、小红书、抖音等非权威信源直接排除

#### 2. 新研究板块审核规则

- **期刊白名单严格执行**：不在名单内的期刊，必须自行核查
- 院校学报需确认是否在CSSCI目录内
- **K12硬边界**：研究对象必须是K12阶段（学前到高中）
  - 高等教育、职业教育、成人教育内容排除
  - 如有K12参考价值，可放入但备注"仅作参考"
- **实践导向优先**：纯理论、模型构建、与教学脱节的研究，优先级"低"或排除
- 关注：教学实践、效果评估、实施策略、案例研究、数据分析、文献综述

#### 3. 新产品板块审核规则

- 只看国外公司
- 优先有融资背景或大公司投资

#### 4. 新报告板块审核规则

- **机构白名单意识**：优先权威机构
- **渠道黑名单**：百度系、东方财富系、营销号、未经认证自媒体
- **追溯原始出处**：有价值信息需找到官方发布页面
- **时效性加严**：政策类只看最近发布，趋势类最多放宽到近一个月

#### 5. 自动标注规则

| 标签 | 触发条件 |
|------|----------|
| ⚠️ 来源待确认 | 来源机构不在白名单且无法100%确认权威性 |
| 🔁 与前期内容相关 | 是此前已推送内容的后续进展 |
| 🌟 编辑推荐 | AI综合判断最值得关注的1-2条 |

### 第二步：人工审核（王老师）

#### 审核列表格式（清单体）

每条资讯包含：
- 序号
- 资讯标题（英文保留+中文翻译）
- 发表时间
- 内容简介（**至少300到400字**）
- 来源机构
- 建议优先级（高/中/低，附100字理由）
- 是否建议深度解读（是/否，附100字理由）

#### 王老师回复格式示例

```
通过项: 1,3,5,7
不通过: 4,6（理由：与K12关联度低）
深度解读: 1,5
```

### 第三步：正式报告生成

#### 正式报告格式

每条资讯包含：
- 序号
- 资讯标题（英文保留+中文翻译）
- 发表时间
- 内容简介（**至少300到400字**，说清前因后果、研究内容与成果、亮点价值）
- 来源机构（用全称）

**注意：**
- 删除"优先级"和"是否深度解读"字段（仅审核列表显示）
- 生成文本版和HTML版

## 四、深度解读报告（额外服务）

### 规格
- 字数：1500字左右
- 不在日报/周报范围内，单独生成
- 开头写明原文标题（英文+中文翻译）
- 文末附原文链接地址

### 结构
调用研究skills进行写作

## 七、时间安排（阶段性通知）

### 核心原则：通知≠等待
**每个阶段完成后：通知王老师 → 立即继续下一步，不等用户回应**

| 阶段 | 任务 | 通知内容 | 通知后行动 |
|------|------|----------|-----------|
| 1 | 搜索完成 | "📊 搜索已完成，共X条" | **立即进入数据处理** |
| 2 | 处理完成 | "📊 数据处理完成，去重后Y条" | **立即生成审核列表** |
| 3 | 审核列表就绪 | "📋 审核列表已准备好，请审核" | **等待王老师审核** ← 唯一等待点 |
| 4 | 审核通过 | "📄 正式报告已推送" | 任务完成 |
