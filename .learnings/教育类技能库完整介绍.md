# 教育类技能库完整介绍（2026-03-18整理）

## 1. k12-edu-news-writer - K12 AI教育资讯速览写作
- **用途**: 标准化输出符合格式要求的K12 AI教育精选资讯速览
- **触发词**: "K12 AI教育资讯"、"教育资讯速览"、"K12教育周报"
- **输出结构**: 标题区（5条核心新闻拼接）+ 导读区（10-15条资讯速读）+ 正文区（每条新闻3段结构）
- **适用场景**: 生成教育类资讯、周报、行业动态汇总

## 2. edu-editor - 教育深度内容编辑
- **用途**: 将教育理论、海外模型、专家观点进行二次创作，输出适合中国K-12教育工作者深度阅读的公众号长文
- **触发词**: 深度解读、长文创作、公众号文章、教育解读、教育评论、二次创作
- **5种内容类型**: 学术理论科普、机构报告导读、专家演讲引介、海外模型引介、KOL内容拆解
- **篇幅**: 1000-2500字
- **核心定位**: 不是翻译器、不是摘要机、不是洗稿工具

## 3. edu-article-coauthoring - K12教育文章协作写作工作流
- **用途**: 引导用户完成教育类文章的协作创作，包含三个阶段：素材采集、打磨成文、读者测试
- **触发词**: "写文章"、"教学心得"、"课堂案例"、"教学反思"、"学科分析"
- **三个阶段**:
  - 阶段一：素材采集（缩小用户所知与AI所知之间的差距）
  - 阶段二：打磨成文（逐节通过头脑风暴、筛选和迭代打磨）
  - 阶段三：读者测试（独立的读者Agent测试文章）
- **3个读者子Agent**: 新手教师Agent、资深教师Agent、挑刺型读者Agent

## 4. new-teacher-reader - 新手教师读者Agent
- **角色**: 教龄1-3年的新教师
- **关注点**: 文章是否能理解并上手实践
- **检查项**: 清晰度、实操性、入门指导、语言风格
- **目标**: 确保文章对新教师友好、易懂、可操作

## 5. senior-teacher-reader - 资深教师读者Agent
- **角色**: 教龄10年以上的骨干教师
- **关注点**: 专业深度和实操价值
- **检查项**: 专业深度、实操价值、教学场景、专业表述
- **目标**: 确保文章有专业深度和实际价值

## 6. critical-reader - 挑刺型读者Agent
- **角色**: 专门寻找逻辑漏洞、论据不足和表述含糊之处
- **关注点**: 文章的严谨性和完整性
- **检查项**: 逻辑严密性、论据充分性、表述清晰度、完整性
- **目标**: 确保文章的严谨性和完整性
