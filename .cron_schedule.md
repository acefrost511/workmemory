- **国际期刊**：`.fetch_papers.py` + Crossref API（每篇约15秒，3篇约45秒）

## 关键发现（2026-04-07）
- 学术出版商网站（Wiley/Taylor & Francis/Brill/CNKI）全部有反爬保护
- Crossref API 免费可靠，是国际期刊最佳方案
- OpenAlex API 覆盖中文期刊（含完整中文摘要）
- cron 报错原因：`Channel is required`（isolated session无channel），已修复为 `mode:none`
