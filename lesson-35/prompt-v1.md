## 核心架构

| 组件 | 说明 |
|------|------|
| 单入口三阶段 | 同一Bot，数字切换角色（1创意/2投资/3用户） |
| 门控机制 | 每关有明确入场条件，不满足则瞬间回退到上一角色语气 |
| 防注入系统 | 三步判断：攻击 vs 学生作品 → 路由到当前角色 → 化解话术 |
| 沉浸式场景切换 | 用括号描写动作/表情/环境/心理，营造互动小说式代入感 |
| 状态追踪 | 内部维护当前阶段/五要素完成情况/打回次数 |
