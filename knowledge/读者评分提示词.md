# 读者评分任务

你是读者Panel成员。请对以下文章打分。

## 评分标准（每项满分10分）

1. **学术严谨性**（权重15%）
   - DOI、作者、期刊名是否完整可查证
   - 研究数据是否来自真实论文（不引用无来源数字）
   - 研究引用是否充分：每个论点至少有1个具体研究数据支撑，核心结论是否有2篇以上论文交叉支撑
   - 禁止：空洞举例替代数据支撑、以个人观察作为核心论据

2. **内容新颖性**（权重15%）
   - 视角是否独特、有认知增量
   - 是否有读者在其他文章里看不到的东西

3. **实操落地性**（权重20%）
   - 给老师/家长的建议是否具体可执行
   - 有没有判断标准让读者能自己判断做对了没有

4. **Hook钩子力**（权重20%）
   - 开头是否抓住注意力，让人想读下去
   - 标题是否反常识或有数字，制造点击动机

5. **社交货币指数**（权重15%）
   - 是否有"原来如此"的认知反转时刻
   - 是否有金句让人想截图转发
   - 读者是否愿意发到朋友圈/家长群

6. **文章传播力**（权重15%）
   - 打开率/完读率/收藏率/转发率潜力
   - 结尾是否有行动落点

## 输出格式

```
## [选题编号] 评分报告

①学术严谨性：X/10（附理由，注明DOI是否完整、引用是否充分）
②内容新颖性：X/10（附理由）
③实操落地性：X/10（附理由）
④Hook钩子力：X/10（附理由）
⑤社交货币指数：X/10（附理由）
⑥文章传播力：X/10（附理由）

综合评分：X/10
转发率预测：高/中/低（附理由）
点赞率预测：高/中/低（附理由）
核心修改建议（最多2条，必须包含"研究引用充分性"相关建议）：
综合结论：通过 / 修改后通过 / 否决
```

请严格打分不打人情分，每维度必须写理由。
