# Impact of generative AI interaction and output quality on university students' learning outcomes

## 基础信息
- **标题**: Impact of generative AI interaction and output quality on university students' learning outcomes: a technology-mediated and motivation-driven approach
- **作者**: Yun Bai (太原工业工程学院), Shaofeng Wang (福州外语外贸学院)
- **期刊**: Scientific Reports (Nature Publishing Group)
- **发表时间**: 2025年7月5日发表; 接收2025年3月12日; 接受2025年6月23日
- **DOI**: 10.1038/s41598-025-08697-6
- **获取**: Open Access (2万次访问, 30次引用, Altmetric 307)
- **URL**: https://www.nature.com/articles/s41598-025-08697-6

## 核心发现

本研究基于互动理论和技术中介学习理论，开发并实证检验了一个综合模型，考察生成式AI（GAI）互动质量、GAI输出质量与学习成果之间的关系，并纳入学习动机、学术自我效能感和创造性自我效能感作为中介变量，以创造性思维作为调节变量。

**研究设计：**
- 样本：323名中国大学生（山西省太原市）
- 方法：两波纵向调查（第一波：2024年6月13-30日；第二波：4个月后）
- 最常用GAI工具：ChatGPT（68.4%受访者使用）

**直接效应（核心数据）：**
- GAI输出质量 → 学习成果：β = 0.346, p < 0.001（第二波数据）
- 学习动机 → 学习成果：显著正向
- 学术自我效能感 → 创造性自我效能感：β = 0.373, p < 0.001
- 学习动机 → 学术自我效能感：β = 0.421, p < 0.001
- 创造性自我效能感 → 学习成果：显著正向

**中介效应：**
- 学习动机中介GAI输出质量与学习成果的关系
- 间接效应：0.085；直接效应：0.453；综合效应：0.538
- 学术自我效能感（β = 0.421）和创造性自我效能感（β = 0.200）也是中介

**创造性思维的调节作用（随时间演变）：**
- 第一轮：创造性思维对GAI输出质量→学习动机的调节：β = 0.535, p < 0.05（正向）
- 第二轮：β = 0.132, p < 0.05（持续正向但减弱）
- 初始阶段：创造性思维对GAI互动→学习动机的调节为负（β = -136.322, p < 0.001），但第二轮变为正向（β = 0.182, p < 0.05）

**核心结论：**
1. GAI互动质量和输出质量通过动机机制显著影响学习成果
2. 互动质量和输出质量对优化学生学习体验都不可或缺
3. 创造性思维的调节作用复杂且随时间演变
4. 当实施恰当时，GAI可增强创造性问题解决表现和学习成果
5. "新奇效应"影响初始参与，但熟悉度随时间改善有效利用
