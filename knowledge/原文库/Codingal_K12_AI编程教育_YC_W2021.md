# Codingal — K-12 AI与编程在线教育平台

**来源**：Y Combinator W2021
**官网**：https://www.codingal.com

## 一、产品概述

Codingal 是全球K-12 AI与计算机科学在线教育的领导者，专注为1-12年级（Grade 1-12）的青少年提供高质量、有认证的个性化编程和AI教育。公司的使命是帮助学生**爱上AI和编程，成为未来的创造者**——无论是企业家、工程师还是科学家。

核心特色：**真人专家教师 + AI辅助学习**的混合模式，所有课程均获得STEM.org认证，并与K-12计算机科学框架对齐。

---

## 二、核心产品与服务

### 2.1 在线直播课程
- **真人专家授课**：所有教师均来自计算机科学背景，经过严格筛选和培训
- **个性化学习路径**：根据学生年龄和技能水平量身定制课程
- **班型灵活**：提供1对1和小组课两种模式，确保个性化关注

### 2.2 编程工具与技术覆盖
| 阶段 | 覆盖工具/语言 |
|------|--------------|
| 启蒙阶段 | Scratch, Minecraft教育版 |
| 进阶阶段 | Python, Roblox, HTML/CSS, JavaScript |
| 高级阶段 | Java, AP计算机科学, 数据科学 |
| AI专项 | ChatGPT等AI工具、机器学习框架 |
| 移动开发 | MIT App Inventor, Thunkable, Pygame |

### 2.3 项目制学习
学生通过动手项目进行学习，包括：
- 开发手机应用（App Inventor）
- 制作游戏（Roblox、Python Pygame）
- 构建网站（HTML/CSS/JavaScript）
- 动画制作

### 2.4 竞赛与营地
- **全球编程竞赛**：包括HPE CodeWars等知名赛事
- **编程夏令营**：培养创造力和自信心的集中学习项目

### 2.5 AI教育专项
随着生成式AI的兴起，Codingal已将AI工具（ChatGPT等）和机器学习基础纳入课程体系，帮助学生在K-12阶段就掌握AI素养。

## 五、商业模式

- **B2C**：家长直接购买课程（1对1或小组课）
- **B2B2C**：与学校、培训机构合作提供课程内容
- **课程包**：按课时或按课程包购买

---

## 六、竞争壁垒与特色

1. **STEM.org认证的权威课程体系**：课程内容经过独立第三方认证，对家长和学校采购具有强说服力
2. **K-12全覆盖**：从小学1年级到高中12年级均有对应课程，且课程随年龄增长动态升级
3. **全球规模验证**：100万+注册学生、4.9/5评分、NPS 86——三项指标均为行业顶尖水平
4. **AI教育先发优势**：在ChatGPT浪潮之前已将AI工具纳入课程体系，是最早面向青少年的AI教育产品之一
5. **混合模式（Human + AI）**：不是纯AI替代教师，而是AI辅助真人教师，既保证了教学质量又实现了规模化
