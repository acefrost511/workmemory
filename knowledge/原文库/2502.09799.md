# Co-designing Large Language Model Tools for Project-Based Learning with K12 Educators

**来源**: arXiv (cs.AI/cs.EDU)
**arXiv ID**: 2502.09799
**提交日期**: 2025-02-13
**作者**: Prerna Ravi, John Masla, Gisella Kakoti, Grace Lin, Emma Anderson, Matt Taylor, Anastasia Ostrowski, Cynthia Breazeal, Eric Klopfer, Hal Abelson
**原文链接**: https://arxiv.org/abs/2502.09799
**DOI 验证**: https://doi.org/10.48550/arXiv.2502.09799

## 核心发现

本文记录了与跨学科K-12教师共同设计LLM工具以支持项目式学习（PBL）实施的合作过程。研究采用参与式设计方法，由MIT媒体实验室团队联合多学科教师，共同探索LLM在支持PBL教学中的实际应用场景和工具需求，为教师赋能的LLM工具开发提供了共创设计框架。
