# Turning Language Model Training from Black Box into a Sandbox

**来源**: arXiv (cs.AI/cs.EDU)
**arXiv ID**: 2601.21631
**提交日期**: 2026-01-29
**作者**: Nicolas Pope, Matti Tedre
**原文链接**: https://arxiv.org/abs/2601.21631
**DOI 验证**: https://doi.org/10.48550/arXiv.2601.21631

## 核心发现

本文开发了一个基于浏览器的工具，允许学生在自己的设备上完全训练一个小型的Transformer语言模型。在CS1课程中，162名学生在前后测试中的解释显示，他们向数据和模型推理的转变显著。研究首次实现了让中学生和大学生亲自动手训练LLM的完整闭环，大幅提升了学习者对语言模型工作机制的理解。
