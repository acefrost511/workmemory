# UNESCO报告：AI Competency Framework for Teachers

## 基础信息
- **标题**: AI Competency Framework for Teachers
- **发布机构**: 联合国教科文组织（UNESCO）
- **发布时间**: 约2024年
- **来源域名**: unesco.org / unesco.org/SDL（UNESCO数字图书馆）
- **报告全文链接**: https://unesdoc.unesco.org/（需通过数字图书馆检索具体文档编号）

## 核心内容

### 报告概述
UNESCO发布的教师AI能力框架，旨在帮助教育工作者理解AI基本概念并合乎伦理、有效地将AI融入课堂教学。

### 主要模块
- AI基础知识
- 数据隐私保护
- 合乎伦理的AI使用指南
- 日常教学中有效应用AI

### 报告地位
UNESCO AI能力框架系列之一，UNESCO还同步发布了针对学生的AI能力框架，共同构成K-12 AI教育政策参考体系。
