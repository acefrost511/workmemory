# 【英国】AI辅导工具政策公告

## 基本信息

| 项目 | 内容 |
|------|------|
| 国家 | 英国 |
| 机构 | 英国教育部（DfE）、科学、创新与技术部（DSIT） |
| 政策名称 | 450,000 Disadvantaged Pupils Could Benefit from AI Tutoring Tools |
| 发布时间 | 2026年1月26日 |
| 官方URL | https://www.gov.uk/government/news/450000-disadvantaged-pupils-could-benefit-from-ai-tutoring-tools |

## 政策概述

英国政府宣布，多达45万名弱势背景学生将从安全的AI驱动辅导工具中受益，获得个性化的一对一学习支持，缩小与同龄人之间的学业差距。相关工具将于2027年底前在学校推出。

## 主要内容

### 核心目标

- 打破机会壁垒，使每个孩子无论背景如何都能成就和成长
- 缩小弱势儿童与同龄人之间的学业差距（目前仅有1/4的弱势学生在GCSE英语和数学中获得5分或以上成绩，而同龄人中超过一半达到这一水平）
- 在学生遇到困难时提供额外帮助，识别需要更多练习的领域
- 补充而非取代高质量的面对面教学
- 为最需要帮助的学生提供有针对性的支持，同时推动最有能力的学生更进一步

### 关键细节

| 项目 | 内容 |
|------|------|
| 受益人数 | 最多45万名免费校餐学生（仅9-11年级即多达） |
| 时间线 | 2026年夏季学期开始教师主导的联合创作；2027年底前工具在学校可用 |
| 资金投入 | £2300万扩展EdTech试验台试点项目（1000+所学校和学院） |
| 试验 | 2026年下半年开始在全国初中生中进行试验 |
| 安全性 | 工具将经过严格测试，与教师共同开发，并与国家课程保持一致 |

### 证据基础

一对一辅导可加速学习约五个月（教育捐赠基金会）。

### 相关措施

- 就儿童社交媒体使用和学校手机禁令进行咨询
- 首次发布5岁以下儿童屏幕使用指南
- 更新教育工具开发技术公司的安全标准
