# FACET: Teacher-Centred LLM-Based Multi-Agent Systems-Towards Personalized Educational Worksheets

**来源**: arXiv (cs.AI/cs.EDU)
**arXiv ID**: 2508.11401
**提交日期**: 2025-08-18
**作者**: Jana Gonnermann-Müller, Jennifer Haase, Konstantin Fackeldey, Sebastian Pokutta
**原文链接**: https://arxiv.org/abs/2508.11401
**DOI 验证**: https://doi.org/10.48550/arXiv.2508.11401

## 核心发现

本文提出FACET框架——一个面向教师、基于LLM的多智能体系统，用于生成个性化课堂教学材料，整合认知和动机维度。该系统让教师能够快速生成符合学生个体需求的差异化学习材料，为教师减负同时提升教学质量。

## 关键词
FACET / LLM / 多智能体 / 教师工具 / 个性化学习材料 / K-12
