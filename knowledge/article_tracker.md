# 文章数据追踪档案

> 建立时间：2026-03-26
> 用途：追踪每篇文章的传播数据，分析创作规律，指导后续选题

---

## 追踪指标说明

| 指标 | 说明 | 记录时间 |
|---|---|---|
| 阅读量 | 公众号后台显示 | 发布后D1/D3/D7 |
| 转发数 | 读者转发的次数 | 同上 |
| 点赞数 | 纯点赞（区别于在看） | 同上 |
| 在看 | 微信首页"在看" | 同上 |
| 收藏数 | 读者收藏次数 | 同上 |
| 打开率 | 阅读/粉丝总数 | D1时计算 |
| 互动率 | (转发+点赞+在看+收藏)/阅读 | D7时计算 |
| 传播系数 | 转发/阅读 | D7时计算 |

## 效果评级标准

- ⭐⭐⭐：爆款（超同类平均3倍以上）
- ⭐⭐：良好（超同类平均1.5倍）
- ⭐：合格（达到平均）
- ❄️：低于预期，需复盘
