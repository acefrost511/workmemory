# Khazix-Skills
[Permalink: Khazix-Skills](https://github.com/KKKKhazix/Khazix-Skills#khazix-skills)
A collection of AI Skills for managing and evolving your skill library. These tools help you create, maintain, and continuously improve AI skills from GitHub repositories.
## Skills Overview
[Permalink: Skills Overview](https://github.com/KKKKhazix/Khazix-Skills#skills-overview)
| Skill | Description |
| --- | --- |
| [github-to-skills](https://github.com/KKKKhazix/Khazix-Skills/blob/main/github-to-skills) | Convert GitHub repos into AI skills automatically |
| [skill-manager](https://github.com/KKKKhazix/Khazix-Skills/blob/main/skill-manager) | Manage skill lifecycle - check updates, list, delete |
| [skill-evolution-manager](https://github.com/KKKKhazix/Khazix-Skills/blob/main/skill-evolution-manager) | Evolve skills based on user feedback and experience |
\* \* \*
## github-to-skills
[Permalink: github-to-skills](https://github.com/KKKKhazix/Khazix-Skills#github-to-skills)
\*\*Automated factory for converting GitHub repositories into specialized AI skills.\*\*
### Features
[Permalink: Features](https://github.com/KKKKhazix/Khazix-Skills#features)
- Fetches repository metadata (README, latest commit hash)
- Creates standardized skill directory structure
- Generates `SKILL.md` with extended frontmatter for lifecycle management
- Creates wrapper scripts for tool invocation
### Usage
[Permalink: Usage](https://github.com/KKKKhazix/Khazix-Skills#usage)
```
/github-to-skills 
```
Or: "Package this repo into a skill: "
### Example
[Permalink: Example](https://github.com/KKKKhazix/Khazix-Skills#example)
```
/github-to-skills https://github.com/yt-dlp/yt-dlp
```
\* \* \*
## skill-manager
[Permalink: skill-manager](https://github.com/KKKKhazix/Khazix-Skills#skill-manager)
\*\*Lifecycle manager for GitHub-based skills.\*\*
### Features
[Permalink: Features](https://github.com/KKKKhazix/Khazix-Skills#features-1)
- \*\*Audit\*\*: Scan local skills folder for GitHub-based skills
- \*\*Check\*\*: Compare local commit hashes against remote HEAD
- \*\*Report\*\*: Generate status report (Stale/Current)
- \*\*Update\*\*: Guided workflow for upgrading skills
- \*\*Inventory\*\*: List all skills, delete unwanted ones
### Usage
[Permalink: Usage](https://github.com/KKKKhazix/Khazix-Skills#usage-1)
```
/skill-manager check # Scan for updates
/skill-manager list # List all skills
/skill-manager delete  # Remove a skill
```
### Scripts
[Permalink: Scripts](https://github.com/KKKKhazix/Khazix-Skills#scripts)
| Script | Purpose |
| --- | --- |
| `scan\_and\_check.py` | Scan directories, parse frontmatter, check remote versions |
| `update\_helper.py` | Backup files before update |
| `list\_skills.py` | List installed skills with metadata |
| `delete\_skill.py` | Permanently remove a skill |

## skill-evolution-manager
[Permalink: skill-evolution-manager](https://github.com/KKKKhazix/Khazix-Skills#skill-evolution-manager)
\*\*Continuously improve skills based on user feedback and conversation insights.\*\*
### Core Concepts
[Permalink: Core Concepts](https://github.com/KKKKhazix/Khazix-Skills#core-concepts)
1. \*\*Session Review\*\*: Analyze skill performance after conversations
2. \*\*Experience Extraction\*\*: Convert feedback into structured `evolution.json`
3. \*\*Smart Stitching\*\*: Persist learned best practices into `SKILL.md`
### Usage
[Permalink: Usage](https://github.com/KKKKhazix/Khazix-Skills#usage-2)
```
/evolve
```
Or: "Save this experience to the skill"
### Workflow
[Permalink: Workflow](https://github.com/KKKKhazix/Khazix-Skills#workflow)
1. \*\*Review\*\*: Agent analyzes what worked/didn't work
2. \*\*Extract\*\*: Creates structured JSON with preferences, fixes, custom prompts
3. \*\*Persist\*\*: Merges into `evolution.json`
4. \*\*Stitch\*\*: Updates `SKILL.md` with learned best practices
### Scripts
[Permalink: Scripts](https://github.com/KKKKhazix/Khazix-Skills#scripts-1)
| Script | Purpose |
| --- | --- |
| `merge\_evolution.py` | Incrementally merge new experience data |
| `smart\_stitch.py` | Generate/update best practices section in SKILL.md |
| `align\_all.py` | Batch re-stitch all skills after updates |

## Installation
[Permalink: Installation](https://github.com/KKKKhazix/Khazix-Skills#installation)
1. Clone this repository:
```
git clone https://github.com/KKKKhazix/Khazix-Skills.git
```
2. Copy desired skills to your Claude skills directory:
```
# Windows
copy /E Khazix-Skills\github-to-skills %USERPROFILE%\.claude\skills\
# macOS/Linux
cp -r Khazix-Skills/github-to-skills ~/.claude/skills/
```
3. Restart Claude to load the new skills.

## How It Works
[Permalink: How It Works](https://github.com/KKKKhazix/Khazix-Skills#how-it-works)
```
+------------------+ +----------------+ +------------------------+
| github-to-skills | --> | skill-manager | --> | skill-evolution-manager|
+------------------+ +----------------+ +------------------------+
| | |
Create new Maintain & Evolve &
skills from update skills improve based
GitHub repos on feedback
```
\*\*The Complete Skill Lifecycle:\*\*
1. \*\*Create\*\*: Use `github-to-skills` to wrap a GitHub repo as a skill
2. \*\*Maintain\*\*: Use `skill-manager` to check for updates and upgrade
3. \*\*Evolve\*\*: Use `skill-evolution-manager` to capture learnings and improve
