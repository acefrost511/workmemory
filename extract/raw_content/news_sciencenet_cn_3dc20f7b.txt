| | 作者：王昊昊 来源：中国科学报 发布时间：2026/3/5 18:35:50 选择字号：小 中  大 | | --- | | |  | | --- | |  | | 陈晓红委员：构建素养导向的校园AI教育标准体系 | |  |     “在人工智能技术深刻重塑教育形态的形势下，应构建素养导向的校园AI教育标准体系，制定《校园人工智能教育应用标准》，研发学科垂直型素养培育AI模型，完善素养导向的智能评价与支持体系，健全AI教育伦理与长效监管机制。”2026年全国两会召开之际，全国政协委员、中国工程院院士、湘江实验室主任陈晓红对记者表示。   陈晓红。受访者 供图  陈晓红认为，现有AI教育产品存在的核心问题包括价值导向偏差，多聚焦于知识灌输与应试提分，对品格塑造、价值观引领、人文素养培育等核心育人目标支撑不足；在教学适配性、内容专业性、伦理安全性、应用规范性等方面缺乏统一标准，无序进入校园导致“千校一面”的泛化应用；对学生创造力、复杂问题解决能力、同理心等核心素养的识别、培养与评价尚缺乏智能技术支撑。  同时，AI教育产品的支撑保障不足，偏远地区学校硬件设施匮乏、师资AI素养薄弱，优质智能教育资源难以惠及，加剧教育公平鸿沟；缺乏长效的AI教育应用跟踪评估机制。  为此，陈晓红建议，由教育部牵头，联动“教、学、管、评、研”全链条，制定《校园人工智能教育应用标准》，明确“素养优先、伦理底线、教学适配”三大核心原则，划定AI在课堂教学中的应用边界，禁止纯粹应试导向的AI产品进入校园。建立多维度标准细则，比如伦理安全方面明确师生信息保护、算法公平性、内容适宜性等红线要求，产品准入方面组建由教育专家、一线教师、监管人员组成的审核团队，教学应用方面制定分学段、分学科的AI素养培育应用指南。  陈晓红建议，支持高校、科研机构与领军企业共建“人工智能+素养培育”创新实验室，聚焦语文、科学等核心学科，研发贴合学科本质与学生成长规律的垂直型AI教学模型，如语文的文化传承智能体、科学的创新思维训练系统等，破解泛化应用难题。鼓励开发融入中华优秀传统文化、地方特色资源的本土化AI教学案例库。  她还建议完善素养导向的智能评价与支持体系，推动AI技术在素养评价中的创新应用；建立“省级引领、区域协同”的AI教育支持机制，设立普惠专项资金，为偏远地区学校标配基础AI体验区；将AI素养纳入教师培训与评价体系，开展“懂技术、懂教育、懂人心”的新型教师培养；健全AI教育伦理与长效监管机制，定期对校园AI产品的应用效果、伦理合规性进行评估，及时发现并解决技术应用中的新问题。  版权声明：凡本网注明“来源：中国科学报、科学网、科学新闻杂志”的所有作品，网站转载，请在正文上方注明来源和作者，且不得对内容作实质性改动；微信公众号、头条号等新媒体平台，转载请联系授权。邮箱：shouquan@stimes.cn。 | |
