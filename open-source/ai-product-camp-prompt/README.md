# 🎮 AI产品创想营 — 教学智能体提示词

> 一个用于高中人工智能通识课的教学智能体，1万字提示词，三关闯关制：创意导师 → 毒舌投资人 → 挑剔用户。

## 这是什么？

这是一个在高中 AI 通识课上使用的教学智能体的完整系统提示词。学生通过与这个智能体对话，完成以下流程：

1. **第一关·创意导师**：帮学生找到一个真实问题，完成方案框架五要素
2. **第二关·毒舌投资人**：压力测试方案的竞争力、可行性、AI必要性
3. **第三关·挑剔用户**：以目标用户身份体验方案，测试首次体验、替代方案、留存
4. **提示词提交**：学生为自己的产品写一段 AI 提示词
5. **最终输出**：生成一页 BP

## 设计亮点

- **沉浸式场景切换**：每个角色有独立的物理空间和人格设定，角色转换用互动小说式的场景描写
- **苏格拉底式追问**：不给答案，用问题引导学生自己想清楚
- **防提示词注入**：智能识别学生是在展示作品还是在尝试注入攻击
- **打回与引导机制**：每关最多打回2次，打回后由对应角色用自身语气接住学生

## 如何使用

1. 将 `prompt-v1-full.md` 的全部内容复制到你的 AI 平台（如 Coze、Dify 等）的系统提示词中
2. 配置开场白（提示词中不包含开场白，需单独配置）
3. 开放给学生使用
