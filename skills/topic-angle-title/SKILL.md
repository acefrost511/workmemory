# 标题选题技能（Topic-Angle Title Skill）

> 从文章核心洞察出发，通过结构化分析生成高点击率、高转发率标题

## 核心原则

**标题不是包装，标题是选角度的决策。选了标题就等于选了"用什么视角讲这篇文章"。**

选角度 = 定胜负。角度选对了，平庸内容也能被点开。角度选错了，好内容也没人看。

---

## 工作流程

### 第一步：文章深度分析

#### 1.1 找文章能撑住的"真冲突"

对比以下 5 个维度，找出文章里天然存在的真实冲突：

| 冲突类型 | 含义 | 示例 |
|---------|------|------|
| **预期 vs 结果** | 读者预期是什么？文章揭示的真实结果是什么？ | 以为精美=有效，实际是陷阱 |
| **常识 vs 真相** | 读者的常识是什么？文章的真相是什么？ | 以为AI让课堂更好看，实际可能拖累学习 |
| **供需落差** | 市场/读者以为需要什么？实际有效的是什么？ | 以为需要更多提示词技巧，实际需要架构认知 |
| **新旧对比** | 旧方法是什么？新发现是什么？ | 旧：靠经验调教AI；新：AI内部有类学校的组织架构 |
| **成本 vs 收益** | 读者付出了什么成本？得到了什么收益？ | 三年学提示词 vs 搞懂一个架构 |

> ⚠️ 只有文章里有支撑的冲突，才是"真冲突"。文章里没有的冲突，叫"伪冲突"——用了就是标题党。

#### 1.2 提取文章能撑住的核心论点

用一句话概括文章的核心发现。
格式：`[主体] + [核心动作] + [关键结果]`
例：Claude Code 源码泄露揭示，AI Agent 的核心能力不来自提示词，而来自记忆组织架构、定期整理机制和多 Agent 协作这三大设计。

---

### 第二步：生成角度（3-5个）

每个角度必须能回答：
1. 这个角度能撑住哪个"真冲突"？
2. 文章里哪段话、哪个案例、哪个数字能兑现这个角度？

格式：
```
角度X：[角度名称]
→ 真冲突：[哪个冲突]
→ 文章兑现点：[具体位置/哪段话/哪个案例]
```

---

### 第三步：标题生成

#### 3.1 标题类型

| 类型 | 公式 | 适合内容 |
|------|------|---------|
| **情绪型** | 制造情绪波动，引发好奇或共鸣 | 颠覆认知、争议性观点 |
| **干货型** | 明确承诺给读者的具体价值 | 操作方法、清单、案例 |
| **混合型** | 情绪+干货结合 | 最容易出爆款 |

#### 3.2 五感标题法

| 感 | 核心 | 示例方向 |
|----|------|---------|
| **痛** | 精准还原读者真实困境 | 备课到深夜、花了钱没效果 |
| **贪** | 承诺一个具体、可感知的结果 | 3个技巧、成绩提升X% |
| **怕** | 揭示正在恶化的趋势或风险 | 正在被淘汰、孩子落后 |
| **好奇** | 制造必须点开才能解开的谜题 | 揭秘/扒开/原来是这样 |
| **惊** | 揭示反常识或出人意料的真相 | 不是……而是…… |

#### 3.3 标题结构

**最优结构：短钩子 + 长钩子**

- **短钩子**（≤8字，视觉冲击）：制造第一印象，打掉读者防御
  - 公式：`情绪词 + 数字/对比/反转`
- **长钩子**（12-20字，认知填充）：接住第一拳，补充背景或悬念
  - 公式：`具体场景 + 可感知的收获`

#### 3.4 四种高阶表达

1. **否定反转式**：`"不是……而是……"`（最经典的反常识结构）
2. **损失具体化**：把"踩坑""代价"变成具体的、可见的
3. **身份代入式**：让目标读者在0.5秒内判断"这说的就是我"
4. **问句悬念式**：问出一个读者想问但没问出来的问题

---

### 第四步：字数硬性规则

**冒号二段式：**
- 短钩子（冒号前）：≤8字
- 长钩子（冒号后）：12-20字
- 总长：≤30字

**无冒号标题：**
- 总长：≤22字

---

### 第五步：输出格式

每个标题必须标注：

```
**标题X：** 《标题内容》
- 对应角度：[从第一步选的角度]
- 真冲突兑现：文章中哪部分能撑住这个标题
- 五感类型：[用哪种感觉]
- 高阶表达：[用哪种表达]
- 总字数：X字
```

**总体要求：**
- 生成 10 个候选标题
- 覆盖所有能撑住的"真冲突"角度
- 情绪型/干货型/混合型都要有
- 最后 Top 3 + 推荐理由

---

## 自查清单

- [ ] 第一步"真冲突"是否在标题生成前完成？
- [ ] 标题是否能撑住对应的"真冲突？（不能撑住=标题党）
- [ ] 短钩子是否≤8字？长钩子是否12-20字？
- [ ] 总字数是否在硬性规则内？
- [ ] 10个标题是否覆盖了不同的角度？

## 禁止

- 用伪冲突制造虚假张力（文章内容撑不住）
- 标题承诺了文章给不了的东西
- 纯正面承诺（"如何做好xxx"缺乏情绪）
- 英文缩写
- 字数超出规则
