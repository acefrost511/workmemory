# K12 AI教育资讯速览写作技能

## 技能基础信息
- **名称**: k12-edu-news-writer
- **适用角色**: K12教育专家、内容创作团队所有Agent
- **用途**: 标准化输出符合格式要求的K12 AI教育精选资讯速览
- **调用场景**: 生成教育类资讯、周报、行业动态汇总时调用

---

## 输出结构总览

1. **标题区**: 5条核心新闻拼接（10-12字/条）
2. **导读区**: 10-15条资讯速读（20-30字/条）
3. **正文区**: 每条新闻3段结构，附原文链接

---

## 输出规范：标题区与导读区

### 1. 标题区（超长标题拼接）
**格式**: 【表情符号】核心新闻1/【表情符号】核心新闻2/【表情符号】核心新闻3/【表情符号】核心新闻4/【表情符号】核心新闻5
**要求**: 
- 精选5条最吸引眼球的核心新闻拼接
- **单条字数10-12字**（更精炼）
- 兼顾信息完整度和高密度感
**排版**: 标题区下方留2行空行+分割线，再进入导读区

### 2. 导读区（资讯速读）
**格式**: 使用 emoji + 一句话新闻形式，列10-15条核心资讯
**示例**:
```
🤖 AI自适应学习帮助低成就学生提分0.45标准差
📚 正确引导下AI写作辅助提升学生兴趣35%
```
**要求**: 每条控制在20-30字，emoji作为分类标签

---

## 输出规范：正文区

### 3. 正文区（模块化新闻块）

#### 每条新闻结构：
**（1）新闻标题**
- 加粗显示
- emoji+具体新闻标题
- 与导读区对应文字完全一致
- 20-30字

**（2）正文内容（自然分段，无方括号标签）**
- **第一段**: 说明信息来源，包含发布机构+日期+文献/报告全称+核心结论
- **第二段**: 细节展开，关键数据加粗，200-400字，短句为主
- **第三段**: 目标用户价值，明确对教师/家长群体的具体指导意义
- **文末附原文链接**: 🔗+链接地址

**排版要求**:
- 同一条新闻的三段内容之间空1行
- 不同新闻之间加分割线+3行空行留白
- 手机端每屏显示2-3段内容，大量留白

---

## 排版与视觉规范

| 规范项 | 要求 |
|--------|------|
| **Emoji分类** | 作为新闻分类标签（🍎/💰/🦞/🤖等） |
| **加粗内容** | 时间、关键数据、文章标题 |
| **链接格式** | 文末用🔗展示来源地址链接 |
| **段落长度** | 200-400字，手机端每屏2-3段 |
| **留白要求** | 大量留白，阅读体验好 |

---

## 📝 固定写作公式

### 快讯标题
[emoji] [主体] [动作] [关键数据/发现]

### 正文开头
据[权威期刊/机构]报道，[主体]进行了[研究动作]，[核心结论发现]。

### 细节展开
[具体发现，200-400字]

### 结尾收尾
对于[教师/家长]群体来说，[具体价值]。
