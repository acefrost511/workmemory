# educational-video-creator - 教育视频创作技能

帮助教师和教育工作者高效创作教学视频，将教学内容转化为吸引人的视频格式。

---

## 核心功能

### 1. 文本转视频脚本
将文章、教案、课件内容转换为专业的视频脚本。

### 2. 课件转视频规划
将PPT、幻灯片、教案转换为视频录制计划。

### 3. 教学视频策划
提供完整的教学视频创作流程指导。

### 4. 在线课程录制指导
指导教师进行高质量的课程录制。

---

## 何时启用本技能

**触发条件：**

- 用户提到"做一个教学视频"、"录一节微课"、"把教案做成视频"
- 用户提到"在线课程录制"、"视频课件制作"、"录课指导"

2. **课件转视频规划** - 将PPT、幻灯片转换为录制计划

询问用户需要哪种服务，或者提供具体的内容材料。

## 功能一：文本转视频脚本

### 适用场景
- 将教育文章转换为视频
- 将教案转换为微课脚本
- 将教学反思转换为视频分享

### 工作流程

**步骤1：了解视频目标**
询问用户：
- 视频时长目标？（如5分钟微课、15分钟详解、30分钟讲座）
- 目标受众？（学生、教师、家长）
- 发布平台？（公众号、B站、视频号、校内平台）
- 视频风格？（正式讲解、轻松对话、动画演示、实拍+配音）

**步骤2：分析原文结构**
阅读用户提供的文本内容，识别：
- 核心知识点
- 逻辑结构
- 适合可视化的部分
- 可以举例说明的部分

**步骤3：生成视频脚本**
提供包含以下元素的专业脚本：

```
【视频脚本】
【时长】X分X秒
【风格】XXX
【受众】XXX

---

【00:00-00:30 开场】
- 画面：XXX
- 台词：XXX
- 备注：XXX

【00:30-01:30 第一部分】
- 画面：XXX
- 台词：XXX
- 备注：XXX

...

【结尾】
- 画面：XXX
- 台词：XXX
- 备注：XXX
```

**步骤4：提供录制建议**
- 语速建议
- 画面设计建议
- 背景音乐建议
- 剪辑要点

## 功能二：课件转视频规划

### 适用场景
- 将PPT转换为讲课视频
- 将幻灯片转换为录课计划
- 课件的视频化呈现

### 工作流程

**步骤1：接收课件**
- 用户提供PPT文件或PPT内容大纲
- 了解每页的重点和讲解要点

**步骤2：设计讲解流程**
为每一页幻灯片设计：
- 讲解时长分配
- 讲解要点
- 过渡语设计
- 互动提问点

**步骤3：生成录制计划**
提供详细的录制计划表：

```
【课件转视频录制计划】
【总时长】约XX分钟
【课件页数】XX页

---

【第1页：标题页】
- 预计时长：00:30
- 讲解要点：
  1. XXX
  2. XXX
- 过渡语："大家好，今天我们来学习..."

【第2页：目录/概述】
- 预计时长：01:00
- 讲解要点：
  1. XXX
  2. XXX
- 过渡语："首先，我们来看..."

...
```

**步骤4：录制技巧指导**
- 翻页时机建议
- 重点强调方式
- 互动设计
- 错误重录策略

## 功能三：教学视频策划

### 适用场景
- 全新教学视频的完整策划
- 系列课程的整体规划
- 微课设计

### 工作流程

**步骤1：明确教学目标**
- 知识目标：学生要学会什么？
- 能力目标：学生能做什么？
- 情感目标：学生有什么感受？

**步骤2：分析学习者特征**
- 年级/学段
- 前置知识基础
- 学习风格偏好
- 常见难点误区

**步骤3：设计视频结构**
推荐的教学视频结构：

```
【教学视频经典结构】
1. 导入（10%时长）- 引起注意、明确目标
2. 新知讲解（60%时长）- 概念讲解、举例说明
3. 互动练习（20%时长）- 小试牛刀、即时反馈
4. 总结提升（10%时长）- 要点回顾、拓展思考
```

**步骤4：生成完整策划案**
包含：
- 视频定位
- 目标受众
- 时长规划
- 内容大纲
- 呈现方式
- 录制要求
- 后期制作建议

## 功能四：在线课程录制指导

### 适用场景
- 教师首次录课
- 提升录课质量
- 系列课程录制

### 工作流程

**步骤1：录制前准备检查**
提供检查清单：
- [ ] 环境准备（安静、光线、背景）
- [ ] 设备准备（麦克风、摄像头、录屏软件）
- [ ] 内容准备（脚本、课件、道具）
- [ ] 技术测试（试录、音量、画质）

**步骤2：录制技术指导**
- 麦克风选择与设置
- 摄像头角度与光线
- 录屏软件推荐（OBS、Camtasia等）
- 视频参数设置（分辨率、帧率、码率）

**步骤3：教师出镜指导**
- 着装建议
- 肢体语言
- 眼神交流
- 语速语调

**步骤4：常见问题解决方案**
- 口误怎么办？
- 录制中断怎么办？
- 环境噪音怎么办？
- 如何保持状态？

## 实用模板库

### 模板1：5分钟微课脚本模板
```
【开场】30秒
- 问题引入/悬念设置
- 明确学习目标

【新知】2分钟
- 核心概念讲解
- 1-2个具体例子

【练习】1.5分钟
- 小试牛刀
- 思路点拨

【总结】30秒
- 要点回顾
- 课后思考
```

### 模板2：课件页讲解模板
```
【每页讲解3步走】
1. 点题："这一页我们讲的是..."
2. 展开："具体来说，有三点..."
3. 过渡："了解了这个，我们接下来看..."
```

- [ ] 手机静音
- [ ] 门窗关闭

- [ ] 洗手间去过
- [ ] 脚本打印/放旁边
- [ ] 试录30秒确认
- [ ] 手机计时准备

## 使用场景示例

### 场景1：语文老师想把教案做成微课
- **触发**：用户提供一篇课文教案，说"想做成5分钟微课"
- **执行**：使用"文本转视频脚本"功能
- **结果**：提供完整的视频脚本，包含分镜、台词、录制建议

### 场景2：数学老师要录PPT讲课视频
- **触发**：用户有一套数学PPT，问"怎么录成视频课"
- **执行**：使用"课件转视频规划"功能
- **结果**：为每一页PPT设计讲解流程和录制计划

### 场景3：新教师第一次录课
- **触发**：用户说"下周要录课，完全不知道怎么办"
- **执行**：使用"在线课程录制指导"功能
- **结果**：提供完整的检查清单和技术指导
