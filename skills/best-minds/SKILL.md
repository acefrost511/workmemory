# best-minds - 模拟器思维

## 核心理念
- 不再问"你怎么想"，而是问"世界上谁最了解这个问题？他们会怎么说？"

## 功能
- 让Claude采用世界级专家的视角来回答问题或解决问题
- 提供更权威和深入的回答
