## 二、PRE-PRODUCTION 策划阶段

### Step 1：确认HKRR定位（每个选题必须过这一关）

**在做任何写作之前，先确认这篇内容对应HKRR的哪个维度：**

| HKRR维度 | 内容要求 | 自查问题 |
|---------|---------|---------|
| H-快乐 | 有情绪体验，不是干巴巴的说教 | 读者读的时候有没有情绪波动？ |
| K-知识 | 有读者不知道的信息或视角 | 读者能带走什么新东西？ |
| R-共鸣 | 触发了"说的就是我"的感觉 | 读者会不会觉得"这就是我遇到的问题"？ |
| R-节奏 | 句子长短交错，有阅读呼吸感 | 读起来累不累？ |

**实际操作：**
- 拿到选题，先问：这个内容最核心对应HKRR哪个维度？
- 如果四个维度都没有，**选题取消**
- 如果主要只有一个维度，写作时想办法补充其他维度

### Step 2：HAMD选题筛选

**H - Hook：设计传播钩子**

动手写之前，先写10个标题备选。

标题备选原则（来自HAMD）：
```
① 标题 = 点进来的理由（一句话价值承诺）
② 标题必须和内容形成记忆闭环
③ 标题要有动词，不能只有名词
```

标题类型参考：
- 痛点型：「还在用AI批作业？老师最容易踩的3个坑」
- 反常识型：「同事追着问：你哪来这么多时间批作业？」
- 利益型：「1个方法，让我每周少改10小时作业」
- 好奇型：「AI进课堂后，学生最该学的能力变了」

**A - Anchor：设计记忆锚点**

在写正文之前，先想好：读者读完这篇，最该记住的那一件事是什么？

锚点类型（选一种）：
- 金句锚点：结尾那句最有力的话
- 故事锚点：那个最具体的场景
- 数据锚点：那个最冲击力的数字
- 反转锚点：那个最出人意料的结论

**M - Mind Map：素材发散再收拢**

先做加法（发散），再做减法（收拢）：

发散阶段——把以下全部写出来（不限格式）：
- 我知道的所有相关素材/案例
- 我想表达的所有观点
- 可能的结构方式
- 标题备选

收拢阶段——确定最终结构：
- 选哪个观点作为核心论点
- 选哪3个素材支撑
- 用什么结构串起来

**D - Different：差异化检验**

**这是整个HAMD最重要也最容易被跳过的一步。**

在动笔之前，必须回答：

```
这篇的差异点是什么？
——同类文章会怎么写这个问题？
——我这篇有什么不同？
```

如果回答不出来，说明选题差异化不够，需要重新思考角度。

差异化来源：
- 视角差异：用老师的视角而不是研究者的视角
- 方法差异：提供了别的地方没有的操作步骤
- 立场差异：有明确的观点，不是两边都说
- 叙事差异：用了真实课堂故事，不是概念堆砌

**差异化自检三问：**
1. 读者为什么要看我这篇，而不是看其他文章？
2. 我这篇的哪个观点是别的地方看不到的？
3. 如果读者只记住一句话，是哪句话？

### Step 3：素材准备

**素材来源优先级（从高到低）：**

1. 陛下方观点/观察（最独特，不可复制）
2. 真实课例/教学故事（次独特，有画面感）
3. 学术研究/论文数据（第三方背书）
4. 热点事件/新闻（时效性强）
5. 工具实测记录（可验证）

**素材入库标准：**
- 每条素材标注：来源 + 核心信息 + 可关联的信念编号
- 素材要有具体细节（时间/数字/场景/人物）
- 避免二手素材（转发三次以上的案例要追源）

### Step 4：标题备选通过

**标题通过条件（主编判断）：**
- 有价值承诺（读者知道看完会得到什么）
- 有具体性（不是泛泛而谈）
- 有动词（不是纯名词堆砌）
- 与内容锚点形成记忆闭环

**标题未通过的标志：**
- 标题很泛（「浅谈AI在教育中的应用」）
- 标题党（承诺了内容实现不了的）
- 没有差异化（「AI时代的教育」太泛）

## 三、PRODUCTION 写作阶段

### Step 5：按模板写作

**模板A：工具实测型（四段式）**

```
【H-Hook开头】：痛点场景引入（100字以内）
- 开头3句必须进场景，不能以"在AI时代"开头

【正文三个要点】：
- 要点1：工具是什么+能解决什么问题
- 要点2：实测过程（数据要真，时间要具体）
- 要点3：优缺点诚实说明（不能只说优点）

【A-Anchor结尾】：
- 一个金句（和标题形成记忆闭环）
- 一个今天就能做的最小步骤

【D-Different验证】：
- 写完后回读：这篇的差异点在哪？
```

**模板B：课例分享型（五段式）**

```
【H-Hook开头】：课程背景一句话

【A-Anchor正文】：
- 设计思路：为什么在这里用AI（不是赶时髦）
- 实施过程：具体到"第几分钟做什么"
- 效果反思：数据要真，失败要诚实写

【D-Different结尾】：
- 可复制要点（3条以内）
- 避坑提示（读者最可能犯的错）
```

**模板C：认知升级型（反转式）**

```
【H-Hook开头】：一个反常识的结论

【A-Anchor正文】：
- 现象描述：具体说明这个现象是什么
- 分析：为什么会出现这个现象
- 观点：我的判断是什么（有立场）

【D-Different结尾】：
- 建设性结论：面对这个变化，老师应该怎么做
```

### Step 6：HAMD写作自查

**写完立即检查：**

| 检查项 | 通过标准 | 未通过的行动 |
|--------|---------|------------|
| Hook到位了吗？ | 开头3句进了场景 | 重写开头 |
| Anchor立住了吗？ | 结尾有金句，读者能记住一件事 | 补/换结尾 |
| M-MindMap用了几个素材？ | 至少2个真实素材 | 补充素材 |
| D-Different能回答吗？ | 能说清楚这篇的不同 | 重写或换角度 |
| HKRR全维度了吗？ | 至少有H/K/R中的一个 | 补情绪/共鸣 |

### Step 7：术语处理

- 首次出现专业术语，必须括号注明含义
- 每千字不超过5个专业术语
- 教育学概念用类比解释（"就像……一样"）

## 四、POST-PRODUCTION 评审阶段

### Step 8：多agent评审

**v1生成后必须经过三角色评审，不达标不直接给陛下：**

```
v1生成
  ↓
资深教师agent评审 → 输出评分+修改意见
  ↓
新手教师agent评审 → 输出评分+修改意见（术语障碍重点检查）
  ↓
主编汇总 → 计算加权综合分
  ↓
≥8分 → 进入Step9
  7-8分 → 主编按意见优化 → v2 → 再次评审 → 通过后进Step9
  <7分 → 重新分析素材 → 调整角度 → 重写
```

**评分体系（详细见EDUCATION_WRITING_SOP_v3.0.md）：**
- 资深教师agent：5维度（专业深度/实操价值/场景适配/专业规范/可操作性）满分50
- 新手教师agent：3维度（可读性/吸引力/阅读体验）满分30
- 主编自评：2维度（主题契合度/传播潜力）满分20

**通过门槛：**
- 综合加权分 ≥ 80分
- 新手教师得分 < 总分33% → 强制返工

### Step 9：陛下终审

主编推送评审通过稿 → 陛下终审 → 通过后进排版

### Step 10：排版发布

md2wechat转换 → 粘贴公众号编辑器 → 发布
发布后：
- 记录发布日期到文章追踪档案
- 记录标题/信念编号/HAMD标签
- 设定D1/D7数据收集提醒

## 五、REVIEW 复盘阶段

### Step 11：数据收集

**时间节点：**
- D1（发布后第1天）：催陛下发数据
- D7（发布后第7天）：再次收集，补充完整

**陛下发送格式：**
```
文章：标题
D1数据：阅读xx/转发x/点赞x/在看x/收藏x
D7数据：阅读xx/转发x/点赞x/在看x/收藏x
```

### Step 12：数据录入

主编（臣）负责：
1. 录入 article_tracker.md
2. 计算打开率/互动率/传播系数
3. 对照预期评级，判断实际评级

### Step 13：单篇复盘

**每篇发布后，臣做一次快速复盘（不超过200字）：**
- 这篇为什么比预期好/差？
- HAMD哪个环节做得对/错？
- HKRR哪个维度缺失？
- 下篇可以怎么用这个教训？

### Step 14：阶段性复盘（5篇一复盘）

积累5篇文章后，臣做一次系统性分析：
- 标题规律：哪类标题打开率高？
- Hook类型效果：痛点型 vs 反常识型 vs 利益型，哪个传播更好？
- 差异点验证：有明确差异点的文章 vs 不清晰的，传播有无差异？
- 素材来源效果：陛下方观点 vs 论文数据，哪类更受欢迎？
- HKRR维度缺失检测：哪些文章只给了K（知识），没给H（快乐）和R（共

鸣）？

### Step 15：SOP迭代

根据复盘结论，动态调整SOP规则：
- 发现某类标题持续低效 → 更新标题制作规范
- 发现某个写作环节薄弱 → 增加检查步骤
- 发现某类内容数据持续好 → 强化此类选题比例

## 六、附录：创作检查清单

### 策划阶段检查（写之前）
- [ ] HKRR定位确认（核心维度是哪个？）
- [ ] 10个标题备选（主编审核通过了吗？）
- [ ] 锚点设计（读者记住的那件事是什么？）
- [ ] 差异化自检（能回答"这篇的不同是什么"吗？）
- [ ] 素材准备（至少2个真实素材？）

### 写作阶段检查（写完之后）
- [ ] 开头3句进了场景，没有"在AI时代"开头
- [ ] 有"我"字，第一人称体验
- [ ] 结尾有金句，和标题形成记忆闭环
- [ ] 给了读者今天就能做的最小步骤
- [ ] 没有纯学术表达（「基于XXX」禁用）
- [ ] 每千字术语不超过5个，首次出现有解释
- [ ] D-Different能回答

### 评审阶段检查
- [ ] 通过资深教师agent评审（≥某分）
- [ ] 通过新手教师agent评审（术语友好）
- [ ] 主编综合分≥80分
- [ ] 陛下终审通过

### 发布阶段检查
- [ ] 已录入文章追踪档案
- [ ] 标题/信念编号/HAMD标签已标注
