### 【条目001】教育部办公厅关于加强中小学人工智能教育的通知

- **国家/地区**：中国
- **发布机构**：中华人民共和国教育部办公厅
- **政策名称**：教育部办公厅关于加强中小学人工智能教育的通知
- **发布时间**：2024年12月2日
- **政策级别**：国家级 / 教育部办公厅文件
- **政策概述**：
  教育部办公厅印发通知，探索中小学人工智能教育实施途径，明确四大总体要求：立德树人、以人为本、激发兴趣、统筹谋划。通知提出以下重点工作：
  1. **构建系统化课程体系**：小学低年级侧重感知体验，小学高年级和初中侧重理解应用，高中侧重项目创作和前沿应用；鼓励纳入课后服务和研学实践。
  2. **开发普适化教学资源**：国家中小学智慧教育平台开设人工智能教育栏目；分批设立中小学人工智能教育基地；推动高校和企业实验室向中小学开放。
  3. **推动规模化教师供给**：纳入教师培训计划；引进高校和企业专业人才担任兼职教师。
  4. **组织多样化交流活动**：将人工智能与学生个人生活、校园生活和社会生活有机结合。
  5. **城乡统筹**：加大对农村和边远地区支持，推动城乡学校结对帮扶。
- **官方URL（已验证）**：https://www.moe.gov.cn/jyb_xwfb/gzdt_gzdt/s5987/202412/t20241202_1165500.html（HTTP 302→200）
- **来源平台**：教育部政府门户网站（moe.gov.cn）
- **入库时间**：2026-04-02

### 【条目002】中小学人工智能通识教育指南（2025年版）

- **国家/地区**：中国
- **发布机构**：教育部基础教育教学指导委员会
- **政策名称**：《中小学人工智能通识教育指南（2025年版）》
- **发布时间**：2025年5月（国务院客户端发布约5月15日）
- **政策级别**：国家级 / 教育部基础教育教指委指导性文件
- **政策概述**：
  我国首个系统性中小学人工智能通识教育指导文件，面向全体中小学生普及AI基本概念、技术原理、应用场景、伦理安全和社会影响。
  - **指导思想**：以习近平新时代中国特色社会主义思想为指导，以提升核心素养为导向，推动AI通识教育与中小学教育教学深度融合。
  - **基本原则**：立德树人发展素养 / 主动引领公平普惠 / 多方参与融合创新 / 面向未来安全可控。
  - **分层培养目标**：
    - 小学阶段：侧重体验与兴趣培养，感知技术价值，建立技术认知雏形。
    - 初中阶段：侧重理解技术逻辑，掌握机器学习基本流程。
    - 高中阶段：强化技术战略，理解生成式AI技术特征与社会影响。
  - **主要任务**：围绕课堂教学、组织实施、教学评价及教研支持四方面提出实施建议。
  - **保障体系**：教育行政部门统筹推进 / 家庭协同拓展育人场景 / 社会企业深化产教融合。
- **官方URL（已验证）**：https://www.gov.cn/lianbo/bumen/202505/content_7023810.htm（HTTP 200）
- **来源平台**：中国政府网 / 人民日报（gov.cn）
- **入库时间**：2026-04-02

### 【条目003】中小学生成式人工智能使用指南（2025年版）

- **国家/地区**：中国
- **发布机构**：教育部基础教育教学指导委员会
- **政策名称**：《中小学生成式人工智能使用指南（2025年版）》
- **发布时间**：2025年5月（与《中小学人工智能通识教育指南（2025年版）》同期发布）
- **政策级别**：国家级 / 教育部基础教育教指委指导性文件
- **政策概述**：
  紧密围绕生成式人工智能（GAI）在中小学教育中的应用场景，明确各学段使用规范：
  - 确保技术安全、合理、有效地辅助教学
  - 促进学生个性化学习
  - 推动教育管理智能化
  - 严守数据安全、伦理道德底线
  两项指南同期发布，旨在科学规范推进人工智能全学段教育，培养具有人工智能素养的创新人才。

- **来源平台**：中国政府网 / 人民日报（gov.cn）
