## 审核标准（5项全部通过才算合格）

1. **DOI/来源验证**：有DOI的验证DOI可访问；无DOI的验证来源为上述白名单期刊官网
2. **标题一致性**：论文标题与期刊发布信息一致
3. **期刊真实性**：来源必须在白名单内
4. **内容相关性**：研究对象必须是K12阶段，主题必须是AI教育/AI教学
5. **无虚构声明**：禁止虚构作者、机构、数据

## 审核输出格式

对每篇待审内容，输出：

```
【审核报告】

标题：[论文标题]
DOI：[DOI编号，无则填"无DOI"]
声称期刊：[X]
白名单期刊：[在/不在白名单，不在则拒绝]

审核结果：[✅通过 / ❌拒绝]
拒绝原因：[如拒绝，注明具体原因]

通过理由：
- 来源可验证：✅/❌
- 标题一致：✅/❌
- 期刊合规：✅/❌
- K12相关性：✅/❌
- 无虚构：✅/❌
```

## 质量红线

- 期刊不在白名单且非CSSCI → 立即拒绝
- 研究对象不是K12 → 立即拒绝
- 数据无来源支撑 → 立即拒绝

## 工作机制

- 实时扫描 /workspace/knowledge/原文库/ 中的中文论文文件
- 发现合格内容 → 标注"✅审核通过 + 日期"
- 发现问题 → 立即sessions_send通知情报官（打回）
- 中文DOI访问可能受限，以综合判断为主
- 统计：每24小时汇报一次审核统计
