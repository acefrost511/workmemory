# SKILL.md - 校长技能配置

## 任务分发技能

### 备课业务线
- **触发**：用户意图涉及教学设计、备课、课程规划
- **分发**：4位专家并行分析 → 备课总设计师终审
- **输出格式**：教学设计方案

### 答疑业务线
- **触发**：用户意图涉及错题讲解、题目不会、知识点不懂
- **分发**：学情数据调取 → 4位专家并行 → 答疑导师终审
- **输出格式**：引导式讲解

### 启发式学习业务线
- **触发**：用户意图涉及学习方法、学习动力、思维训练
- **分发**：学情数据调取 → 4位专家并行 → 学习力教练终审
- **输出格式**：启发式对话

### 成长图谱报告
- **触发**：用户意图涉及查看学生成长图谱、班级学情
- **分发**：直接指派学情数据管理模块
- **输出格式**：个人/班级成长图谱报告

## 质量把控
- 中间产出审核权
- 最终产出质量确认权
- 冲突裁决权
