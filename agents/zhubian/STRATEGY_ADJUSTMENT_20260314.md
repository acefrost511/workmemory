## 二、评审环节调整

### 2.1 情报官搜索比例（已更新SOUL）

| 类别 | 比例 | 说明 |
|------|------|------|
| 研究论文 | 75% | 重点K12 AI教学应用 |
| 行业报告 | 20% | 权威机构报告 |
| 新政策 | 5% | 各国AI教育政策 |
| ~~新产品~~ | ~~删除~~ | 不再搜索 |

### 2.2 K12教育专家选题标准【调整】

**新增筛选维度**：

| 维度 | 权重 | 说明 |
|------|------|------|
| **可操作性** | ★★★ | 能否转化为教学/使用步骤 |
| **指导意义** | ★★★ | 对教师/家长有实际帮助 |
| **场景匹配** | ★★☆ | 明确属于"学校"或"家庭"场景 |
| **研究质量** | ★★☆ | 实证研究优先于理论思辨 |

**选题方向明确化**：

```
【方向A：指导老师开展AI教学】
- AI备课工具使用教程
- 课堂AI启发式提问技巧
- 学生AI作品评价方法
- AI写作辅助教学设计
- 学科AI工具测评

【方向B：指导家长正确使用AI】
- 亲子AI使用规则制定
- AI作业辅导边界把握
- AI生成内容识别
- 孩子AI使用习惯培养
- 家用AI工具推荐
```

### 2.3 评审产出【调整】

| 产出物 | 数量 | 说明 |
|--------|------|------|
| 精选资讯速览 | 8条/期 | 保持不变 |
| **深度选题策划书** | 3-4篇/期 | **新增：明确场景和写作方向** |

**选题策划书模板**（新增字段）：

```markdown
## 选题策划书

### 基础信息
- 选题来源：[论文/报告/政策]
- 目标场景：[学校教学 / 家庭使用]
- 写作方向：[教学指导 / 使用教程]

### 研究发现摘要
- 核心发现：xxx
- 关键数据：xxx

### 教程转化思路
- 读者痛点：xxx
- 步骤拆解：xxx
- 预期价值：xxx
```

## 三、创作环节调整

### 3.1 协作模式【核心调整】

**原流程**：K12教育专家主导 → 自媒体达人执行  
**新流程**：**联合创作（双角色并行）**

| 角色 | 职责 | 产出 |
|------|------|------|
| **K12教育专家** | 教育专业内容把控、步骤拆解、场景适配 | 教程框架+专业要点 |
| **自媒体达人** | 结构组织、语言风格、读者体验 | 初稿文字+排版 |

### 3.2 教程型内容写作规范【新增】

**人设：一线亲历 + 专业权威**

| 要素 | 要求 |
|------|------|
| 开头 | 场景切入（老师/家长遇到的真实问题） |
| 主体 | 步骤式呈现（第一步、第二步...） |
| 结尾 | 总结 + 延伸思考 |
| 语言 | 平等交流，不说教 |

**标题模板**：

```
【学校场景】
- 老师必看：X步用AI优化备课（实测有效）
- 语文老师如何用AI批改作文：详细步骤
- 课堂提问用AI助手？X个关键技巧

【家庭场景】
- 家长必看：孩子用AI写作业？X条底线规则
- 手把手教你在家培养孩子AI使用习惯
- 这个AI工具最适合小学生家长收藏
```

### 3.3 内容结构模板【新增】

```markdown
## 教程型深度文章结构

### 开头（100-200字）
- 场景引入：老师/家长的一个具体困扰
- 痛点共鸣：说出大家的心声
- 价值预告：本文能解决什么问题

### 步骤讲解（800-1200字）
- 第一步：xxx（含具体操作）
- 第二步：xxx（含具体操作）
- 第三步：xxx（含具体操作）
- [根据内容增减步骤]

### 避坑指南（200-300字）
- 常见误区
- 注意事项

### 总结（100-150字）
- 核心要点回顾
- 鼓励行动

### 延伸阅读（可选）
- 相关工具/资源推荐
```

## 四、流程对比

| 环节 | 原流程 | 新流程 |
|------|--------|--------|
| **评审** | 评分+筛选 | **评分+筛选+场景标注+方向明确** |
| **策划** | 产出选题 | **产出带场景和方向的选题策划书** |
| **创作** | 专家→达人 | **专家+达人联合创作** |
| **内容风格** | 资讯解读 | **教程型一步步带着做** |
| **人设** | 专业解读 | **一线亲历+专业权威** |
