# Agent 配置说明

## 搜索结果结构比例（铁律）
- **论文研究**：≥70%
- **报告**：≈20%
- **产品**：≈10%

## 论文类型要求（占论文50%以上）
- **数据驱动研究**：基于实验、调查、实证数据分析
- **案例研究**：学校、区域、产品的具体应用案例（含数据）
- **对比研究**：不同教学方法、技术工具的效果对比
- **元分析**：对多项研究的系统综述
- **理论思辨类**：≤50%
