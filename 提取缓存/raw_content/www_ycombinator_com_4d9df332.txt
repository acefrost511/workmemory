### Anyone can create real games with AI
CodeWisp lets anyone build and publish web games using plain-English prompts.
Traditional game development requires learning complex engines, managing assets and code, and following rigid workflows, making it slow and inaccessible for most people.
With CodeWisp, users describe the game they want. CodeWisp generates the game’s code, structure, and assets, and users can refine it with follow-up prompts. This makes it possible to go from idea to a playable web game in minutes, without prior game-development experience.

CodeWisp - AI Game Creation Platform
[See original launch post](/launches/Pbd-codewisp-ai-game-creation-platform)
Hey everyone, I’m [Elvin Fu](https://www.linkedin.com/in/fuelvin/) - founder of [CodeWisp](http://codewisp.ai).
\*\*TL;DR\*\*
\*\*CodeWisp lets anyone create and publish web games using prompts.\*\* Describe the game you want and CodeWisp generates it - no need to juggle code files, engines, or asset pipelines.
\*\*Why this matters:\*\*
Gaming is already a massive industry - it’s expected to grow to nearly \*\*$300B globally by 2029\*\*\*(PwC Global Entertainment & Media Outlook)\*.
But the tools used to build games are still designed for professional studios.
AI is creating a new paradigm where games can be \*\*generated, modified, and shared instantly,\*\* opening game creation to millions more people.
That is the opportunity we’re building toward at CodeWisp.
\*\*The Goal:\*\*
Remove the friction of traditional game development and make creating and publishing games accessible to anyone, not just engineers or studios.
\*\*The Product:\*\*
CodeWisp has two main parts (everything is in browser, no downloads or tools required):
1 ) The Editor
A browser-based workspace where you describe the game you want and CodeWisp generates it. You can prompt edits to add mechanics, enemies, physics, levels, or visuals and see the game update instantly.
CodeWisp can generate 2D games, 3D games, and multiplayer browser games.
![](/media/?type=post&id=98433&key=user\_uploads%2F892590%2Fa884349c-6658-45ae-b86d-65a6cf2c9129&variant=large)
2 ) The Homepage
A place to explore and play games created by the community. Every game can be opened, played in the browser, and shared instantly with a link.
![](/media/?type=post&id=98433&key=user\_uploads%2F892590%2Fb15e27c9-a0e5-462e-b22f-bd07fc362c24&variant=large)
\*\*Traction:\*\*
• 2,000+ weekly active game creators
• 17,000+ games created on CodeWisp
\*\*The Founder:\*\*
Elvin Fu - Making games since age 10, created mobile and web games played by millions. Taught game development to 22+ million people on [YouTube](http://youtube.com/c/fuelvin). Built two game engines from the ground up.

\* If you know someone who’s always wanted to make a game, feel free to send them CodeWisp
